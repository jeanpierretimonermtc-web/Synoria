# Benchmark concurrentiel - Synoria

Date de recherche : 23 juin 2026

## 1. Objectif du document

Ce document sert a comparer Synoria aux logiciels et plateformes deja presents sur le marche, afin de repondre a trois questions :

- les concurrents font-ils mieux que Synoria ?
- sur quels points Synoria est deja fort ?
- que faut-il ameliorer pour se differencier clairement ?

Le benchmark distingue plusieurs familles de concurrents, car Synoria ne rentre pas dans une seule case. Il touche a la fois au dossier patient, a l'agenda, a la facturation, a la comptabilite, au RGPD et aux plugins par specialite.

## 2. Positionnement actuel de Synoria

Synoria est un logiciel de gestion de cabinet pour praticiens independants, avec une logique de dossier patient local et personnalisable.

Ses piliers actuels :

- dossier patient ;
- historique des seances ;
- anamnese de base ;
- formulaire MTC integre ;
- plugins par specialite ;
- agenda et rendez-vous ;
- facturation ;
- comptabilite simplifiee ;
- sauvegardes chiffrees ;
- fonctions RGPD ;
- recherche globale ;
- exports Excel / JSON / PDF ou impression ;
- stockage local ;
- version portable possible.

Synoria n'est donc pas seulement un agenda, ni seulement un dossier patient. C'est plutot un logiciel de cabinet local, specialise et extensible.

## 3. Les familles de concurrents

### Famille A - Plateformes de rendez-vous medical

Exemples :

- Doctolib ;
- Maiia ;
- CLICKDOC ;
- KelDoc / anciens acteurs similaires.

Ces outils sont tres forts pour :

- la visibilite patient ;
- la prise de rendez-vous en ligne ;
- les rappels automatiques ;
- les agendas partages ;
- la teleconsultation ;
- l'image de marque.

Ils sont moins centres sur :

- l'anamnese ultra-specialisee ;
- la personnalisation metier fine ;
- le stockage local ;
- les pratiques complementaires non medicales ;
- la propriete directe des donnees par le praticien hors cloud.

Conclusion pour Synoria : il ne faut pas essayer de battre Doctolib sur la prise de rendez-vous publique. Il faut se differencier sur le dossier patient metier, la personnalisation, les plugins et les donnees locales.

### Famille B - Logiciels de cabinet / practice management

Exemples :

- Cliniko ;
- Jane App ;
- Weda ;
- VEGA.

Ces outils sont plus proches de Synoria car ils gerent le cabinet, les dossiers, les rendez-vous et parfois la facturation.

Ils sont souvent forts sur :

- maturite produit ;
- interface web accessible partout ;
- multi-praticien ;
- portail patient ;
- paiements ;
- teleconsultation ;
- automatisations ;
- support structure ;
- mises a jour continues.

Synoria peut se distinguer par :

- son fonctionnement local ;
- son absence de cloud obligatoire ;
- sa simplicite pour un praticien solo ;
- ses plugins metier ;
- une tarification plus accessible ;
- une approche plus proche des therapeutes independants.

### Famille C - Agendas et prise de rendez-vous generiques

Exemples :

- Calendly ;
- SimplyBook.me ;
- Google Calendar ;
- Reservio ;
- Setmore.

Ces outils sont forts pour :

- prendre des rendez-vous en ligne ;
- envoyer des rappels ;
- synchroniser des calendriers ;
- encaisser des paiements ;
- afficher une page de reservation.

Mais ils ne remplacent pas Synoria car ils ne gerent pas vraiment :

- le dossier patient complet ;
- l'anamnese ;
- le suivi therapeutique ;
- les donnees sensibles ;
- les fiches de seance ;
- le RGPD cabinet ;
- les plugins par specialite.

Conclusion pour Synoria : ces outils sont des complements ou des concurrents partiels, pas des concurrents directs.

### Famille D - Marketplaces de praticiens bien-etre / medecines douces

Exemples :

- Medoucine ;
- Resalib ;
- Liberlo ;
- Therapeutes.com.

Ces plateformes apportent surtout :

- visibilite ;
- referencement ;
- prise de rendez-vous ;
- acquisition de nouveaux clients ;
- image de reseau.

Elles ne sont pas d'abord des logiciels de dossier patient local. Elles peuvent etre utiles pour trouver des patients, mais elles ne remplacent pas un outil interne de suivi clinique.

Conclusion pour Synoria : ne pas se positionner comme annuaire. Se positionner comme le logiciel de travail quotidien du praticien.

## 5. Analyse concurrent par concurrent

### Cliniko

Cliniko est probablement l'un des concurrents internationaux les plus interessants a etudier.

Il propose :

- agenda ;
- reservation en ligne ;
- notes de traitement ;
- dossiers sante ;
- facturation ;
- paiements ;
- rapports ;
- stockage cloud ;
- support ;
- essai gratuit de 30 jours.

Prix public releve :

- 45 $/mois pour 1 praticien ;
- 95 $/mois pour 2 a 5 praticiens ;
- 145 $/mois pour 6 a 8 praticiens ;
- 195 $/mois pour 9 a 12 praticiens ;
- SMS a 10 cents par message.

Ce que Cliniko fait mieux que Synoria aujourd'hui :

- maturite commerciale ;
- site tres clair ;
- essai gratuit ;
- documentation produit ;
- cloud multi-appareil ;
- support structure ;
- gestion multi-praticien ;
- rapports plus avances ;
- API et integrations.

Ce que Synoria peut faire mieux :

- donnees locales ;
- pas de dependance a un cloud ;
- approche francaise ;
- plugins par specialite ;
- MTC tres specialise ;
- possibilite de vendre un pack metier au lieu d'un abonnement mensuel ;
- adaptation fine aux praticiens independants francophones.

Verdict : Cliniko est plus mature comme produit SaaS. Synoria peut se differencier par le local, le francais, les plugins et la specialisation.

### Jane App

Jane App est une solution tres avancee de gestion de clinique et cabinet.

Elle propose :

- agenda ;
- booking en ligne ;
- telehealth ;
- formulaires d'admission ;
- documentation clinique ;
- paiement ;
- facturation ;
- portail patient ;
- messagerie securisee ;
- rapports ;
- IA scribe en option ;
- support humain.

Prix public releve :

- Balance : 54 $/mois ;
- Practice : 79 $/mois ;
- Thrive : 99 $/mois ;
- IA Scribe illimite : 15 $/mois par praticien ;
- telehealth groupe : 15 $/mois par praticien ;
- site web Jane : 59 $/mois.

Ce que Jane fait mieux :

- experience tres complete ;
- portail patient ;
- paiements integres ;
- formulaires en ligne ;
- teleconsultation ;
- IA de notes ;
- image tres professionnelle ;
- plans tarifaires clairs.

Ce que Synoria peut faire mieux :

- beaucoup plus simple pour praticien solo ;
- meilleur controle local des donnees ;
- moins cher si achat unique ;
- plugins metier francophones ;
- meilleur ancrage dans MTC / osteopathie / kinesiologie / naturopathie ;
- version portable.

Verdict : Jane App est tres forte. Synoria ne doit pas la copier integralement, mais peut s'en inspirer pour le site commercial, les formulaires et l'experience de prise en main.

### VEGA

VEGA est un concurrent francais important dans l'univers paramedical, surtout pour les auxiliaires medicaux.

Il met en avant :

- gestion administrative ;
- teletransmission ;
- suivi des patients ;
- gestion comptable ;
- assistance telephonique ;
- mises a jour ;
- outils metier ;
- compatibilite Carte Vitale / CPS selon offres ;
- solution VEGA Tab.

Prix public releve :

- offre IDEL : a partir de 45 €/mois ;
- offre kinesitherapeute : a partir de 45 €/mois ;
- offre orthophoniste : a partir de 41,50 €/mois ;
- offre orthoptiste : a partir de 45 €/mois.

Ce que VEGA fait mieux :

- ancrage reglementaire fort ;
- teletransmission ;
- support France ;
- notoriete ;
- cible paramedicale claire ;
- comptabilite plus avancee pour professions reglementees.

Ce que Synoria peut faire mieux :

- plus adapte aux therapeutes non conventionnes ;
- pas centre sur la teletransmission ;
- plus souple pour les anamnese longues ;
- plugins par specialite ;
- donnees locales/portables ;
- interface plus simple si le besoin est le suivi patient et non la facturation Securite sociale.

Verdict : VEGA est fort pour les professions conventionnees. Synoria doit eviter ce terrain et viser les praticiens qui n'ont pas besoin de teletransmission mais veulent un vrai dossier patient.

### Weda

Weda est un logiciel medical en ligne francais, filiale du groupe VIDAL.

Il met en avant :

- dossier patient ;
- prescription ;
- gestion des rendez-vous ;
- admin et facturation ;
- coordination des soins ;
- interop avec outils metier ;
- hebergement en France ;
- certification HDS ;
- formulaires parametrables ;
- intelligence medicale / Nabla selon modules.

Ce que Weda fait mieux :

- credibilite medicale ;
- HDS ;
- interop ;
- prescription ;
- outils pour medecins ;
- ecosysteme VIDAL ;
- adaptation aux obligations des professions reglementees.

Ce que Synoria peut faire mieux :

- simplicite ;
- prix potentiellement plus accessible ;
- usage hors medecine conventionnelle ;
- pas de cloud obligatoire ;
- plugins tres pratiques ;
- version portable ;
- langage plus proche des therapeutes independants.

Verdict : Weda est une reference medicale, mais probablement trop medicale et trop lourde pour une grande partie de la cible Synoria.

### Doctolib

Doctolib est le leader francais de la prise de rendez-vous medicale.

Ses forces :

- visibilite patient enorme ;
- prise de rendez-vous en ligne ;
- rappels ;
- teleconsultation ;
- marque connue ;
- confiance grand public ;
- ecosysteme patient.

Ses limites par rapport a Synoria :

- pas un outil centre sur l'anamnese metier locale ;
- dependance plateforme ;
- cible surtout professionnels de sante encadres ;
- moins adapte aux praticiens exclus ou mal servis par les grandes plateformes ;
- pas un logiciel local de travail clinique independant.

Verdict : Doctolib n'est pas a affronter frontalement. Synoria peut meme etre complementaire : Doctolib pour trouver/prendre les RDV, Synoria pour travailler le dossier patient.

SimplyBook.me est un outil de prise de rendez-vous et reservation en ligne.

Prix public releve :

- Standard : 29,90 $/mois en mensuel ;
- Premium : 59,90 $/mois en mensuel ;
- prix plus bas en facturation annuelle.

Forces :

- booking en ligne ;
- site de reservation ;
- widget ;
- paiements ;
- rappels ;
- options nombreuses ;
- plan gratuit.

Limites :

- pas concu comme dossier patient complet ;
- pas d'anamnese specialisee ;
- pas de logique therapeutique ;
- pas de stockage local ;
- pas de module MTC/osteo/kinesio.

Verdict : concurrent sur l'agenda uniquement. Synoria peut proposer une integration ou garder son propre agenda simple.

### Calendly

Calendly est un outil de planification tres populaire.

Prix public releve :

- Free : gratuit ;
- Standard : 10 $/utilisateur/mois en annuel ;
- Teams : 16 $/utilisateur/mois en annuel ;
- Enterprise : a partir de 15 000 $/an.

Forces :

- simplicite ;
- partage de lien ;
- integrations Google/Outlook/Teams/Zoom ;
- paiement Stripe/PayPal ;
- rappels ;
- automatisations.

Limites :

- pas de dossier patient ;

- pas d'anamnese ;
- pas de facturation cabinet complete ;
- pas de plugins de specialite.

Verdict : outil excellent pour prendre des rendez-vous, mais pas un vrai concurrent coeur.

### Medoucine et marketplaces similaires

Medoucine, Resalib, Liberlo et plateformes similaires servent surtout a donner de la visibilite aux praticiens de medecines douces ou d'accompagnement.

Forces :

- acquisition de clients ;
- annuaire ;
- prise de rendez-vous ;
- image specialisee bien-etre ;
- rassurance patient via reseau.

Limites :

- dependance a une plateforme ;
- pas de logiciel local ;
- peu ou pas de dossier patient metier complet ;
- prix parfois peu visible publiquement ;
- logique marketing plus que logique de suivi clinique.

Verdict : ce ne sont pas des concurrents directs du coeur Synoria. Ce sont des concurrents du budget marketing du praticien.

## 6. Synoria est-il mieux ou moins bien ?

### Synoria est deja meilleur sur ces points

- Donnees locales.
- Version portable possible.
- Plugins par specialite.
- Anamnese MTC tres specialisee.
- Possibilite de plugin osteopathie, naturopathie, kinesiologie.
- Simplicite pour praticien solo.
- Sauvegardes chiffrees.
- RGPD integre.
- Facturation et comptabilite simplifiee dans le meme outil.
- Potentiel d'achat unique, plus rassurant que l'abonnement mensuel.

### Les concurrents sont meilleurs sur ces points

- Notoriete.
- Site commercial.
- Onboarding.
- Essai gratuit.
- Mises a jour automatiques.
- Support client structure.
- Multi-appareil cloud.
- Portail patient.
- Paiements en ligne.
- Rappels SMS.
- Application mobile.
- Documentation marketing.
- Integrations externes.
- IA de synthese ou de prise de notes.

### Conclusion honnete

Synoria a un vrai angle different, mais il doit etre mieux emballe commercialement. Le produit a deja des briques solides ; ce qui manque le plus n'est pas seulement technique, c'est la presentation, l'onboarding, la confiance et la lisibilite de l'offre.

## 7. Opportunites de differenciation

### 1. Devenir le logiciel local des praticiens independants

Phrase possible :

> Vos dossiers patients restent chez vous, pas dans un cloud impose.

Ce positionnement est clair et distinctif.

### 2. Vendre des packs par specialite

Exemples :

- Synoria MTC ;
- Synoria Osteopathie ;
- Synoria Kinesiologie ;
- Synoria Naturopathie ;
- Synoria Praticien bien-etre.

Chaque pack doit avoir :

- son plugin ;
- une page commerciale dediee ;
- des captures ;
- une fiche de fonctionnalites ;
- un prix clair ;
- un guide d'installation.

### 3. Mettre en avant l'anamnese specialisee

La plupart des concurrents disent "formulaires personnalisables". Synoria peut dire :

> Des fiches metier deja pretes, construites pour votre pratique.

Cela parle plus aux praticiens.

### 4. Miser sur le "tout-en-un sobre"

Promesse :

> Patients, seances, agenda, factures, compta et sauvegardes dans un seul outil simple.

### 5. Proposer un tarif lisible

Les praticiens solos aiment savoir combien ils paient.

Options recommandees :

| Offre | Prix possible | Commentaire |
|---|---:|---|
| Synoria Core | 149 a 199 € achat unique | logiciel de base sans plugin specialise |
| Pack specialite | 199 a 249 € achat unique | Core + 1 plugin metier |
| Plugin supplementaire | 49 a 99 € | utile pour multi-pratiques |
| Maintenance/support annuel | 69 a 99 €/an | mises a jour + support email |
| Abonnement alternatif | 12 a 19 €/mois | pour ceux qui preferent lisser |

Le prix doit rester sous le cout annuel de Cliniko/Jane/VEGA pour etre attractif.

## 8. Priorites produit pour battre les concurrents

### Priorite 1 - Site commercial clair

Creer une page qui explique :

- pour qui est Synoria ;
- ce qu'il fait ;
- pourquoi les donnees locales sont un avantage ;
- quelles specialites sont couvertes ;
- combien ca coute ;
- comment essayer.

### Priorite 2 - Demo video

Une video de 3 a 5 minutes :

- creer un patient ;
- faire une seance ;
- utiliser un plugin ;
- generer une facture ;
- sauvegarder.

### Priorite 3 - Essai gratuit

Un essai de 14 ou 30 jours rassurerait beaucoup.

### Priorite 4 - Mise a jour automatique

Les concurrents SaaS sont toujours a jour. Pour compenser le local, Synoria doit simplifier les mises a jour.

### Priorite 5 - Rappels SMS/email

Pour rivaliser avec les agendas, il faudra a terme :

- email de rappel ;
- SMS optionnel ;
- message personnalisable ;
- confirmation de RDV.

### Priorite 6 - Portail ou formulaire patient avant consultation

Un formulaire d'anamnese pre-consultation envoye au patient serait une grosse avancee :

- gain de temps ;
- image professionnelle ;
- meilleur onboarding ;
- donnees structurees.

### Priorite 7 - Templates de seance

Les praticiens repetent souvent des structures. Des modeles pre-remplis seraient tres utiles :

- premiere consultation ;
- suivi ;
- bilan osteo ;
- consultation MTC ;
- bilan naturopathie ;
- seance kinesiologie.

## 9. Angle marketing recommande

Ne pas dire seulement :

> Logiciel de dossier patient.

Dire plutot :

> Synoria est le dossier patient local et personnalisable des praticiens independants.

Ou :

> Un logiciel simple pour suivre vos patients, structurer vos seances et garder vos donnees chez vous.

Ou encore :

> L'alternative locale aux plateformes cloud pour les therapeutes et praticiens independants.

## 10. Ce que Synoria ne doit pas essayer de devenir tout de suite

Synoria ne doit pas chercher immediatement a devenir :

- un Doctolib bis ;
- un logiciel de teletransmission ;
- un logiciel medical hospitalier ;
- un SaaS complet multi-clinique ;
- une plateforme marketplace ;
- un outil d'IA medicale decisionnelle.

Ce serait trop lourd, trop cher et trop reglementaire.

Le bon chemin est :

1. logiciel local solide ;
2. plugins metier excellents ;
3. experience commerciale rassurante ;
4. automatisations utiles ;
5. integrations legeres.

## 12. Recommandation strategique

La meilleure strategie n'est pas de copier les gros concurrents. Elle est de devenir excellent sur une niche claire :

> les praticiens independants qui veulent un dossier patient local, simple et adapte a leur specialite.

Pour cela, Synoria doit prioriser :

1. une anamnese de base impeccable ;
2. des plugins tres qualitatifs ;
3. un site de vente rassurant ;
4. une demo video ;
5. un prix simple ;
6. un installateur signe ;
7. un essai gratuit ;
8. une mise a jour plus fluide ;
9. une documentation utilisateur professionnelle.
