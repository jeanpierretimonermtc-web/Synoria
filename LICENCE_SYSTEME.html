Documentation technique & procédure d'implémentation Version 1.0 — Juin 2026

1. Vue d'ensemble

Le système de licence proposé fonctionne 100% hors-ligne (offline). Il ne nécessite aucun serveur et valide chaque clé par signature cryptographique HMAC-SHA256.

CritèreValeur
AlgorithmeHMAC-SHA256 (Node.js natif)
Dépendances supplémentairesAucune (crypto intégré)
Serveur requisNon — validation locale uniquement
Connexion internet requiseNon
Stockage licenceuserData/license.json
Clé secrèteDans le main process uniquement (jamais exposée)
Mode trial30 jours à partir du 1er lancement
Principe : Tu génères les clés avec un script privé (sur ton PC uniquement). L'app vérifie la signature au démarrage sans contacter Internet. Si la signature est valide → l'app s'ouvre. Sinon → écran d'activation ou message trial expiré.

2. Format de la clé de licence

Format : MTC1-{PAYLOAD_BASE64}-{SIGNATURE_HEX_8}

MTC1-eyJuYW1lIjoiSmVhbi1QaWVycmUiLCJlbWFpbCI6ImV4YUBleGFtcGxlLmNvbSIsImV4cGlyeSI6IjIwMjctMDYtMTIiLCJlZGl0aW9uIjoicHJvIn0=-a3f9b2c1

Le PAYLOAD est un JSON encodé en Base64 contenant :

{
  "name":    "Jean Dupont",          // Nom du praticien
  "email":   "contact@cabinet.fr",   // Email d'achat
  "issued":  "2026-06-12",           // Date d'émission
  "expiry":  "2027-06-12",           // Date d'expiration (ou "perpetual")
  "edition": "pro",                  // "trial" | "standard" | "pro"
  "seats":   1                       // Nombre de postes autorisés
}

La SIGNATURE est les 8 premiers caractères du HMAC-SHA256 du payload avec ta clé secrète. C'est ce qui garantit que la clé n'a pas été falsifiée.

function generateLicense(payload, secret) {
  const payloadStr  = JSON.stringify(payload)
  const payloadB64  = Buffer.from(payloadStr).toString('base64')
  const sig         = crypto.createHmac('sha256', secret)

                            .digest('hex')

  return `MTC1-${payloadB64}-${sig}`
}

6. Code — Service de validation (dans l'app)

src/main/services/licenseService.ts
import * as crypto from 'crypto'
import * as fs from 'fs'
import * as path from 'path'
import { app } from 'electron'

// ⚠ Même clé que dans generate-license.cjs — main process uniquement
const HMAC_SECRET = 'e418dbf96d47f7a2e199b34d89c5933e7aa1f5dca99206f2d56de4ed45b1ef10'
const TRIAL_DAYS  = 30

export interface LicenseInfo {
  valid:     boolean
  mode:      'licensed' | 'trial' | 'expired' | 'invalid'
  name?:     string
  email?:    string
  edition?:  string
  expiry?:   string
  trialDaysLeft?: number
  daysLeft?:      number
}

// ─── Chemin du fichier licence utilisateur ───────────────────────────────────
function getLicensePath(): string {
  const base = process.env.PORTABLE_EXECUTABLE_DIR
    ? path.join(process.env.PORTABLE_EXECUTABLE_DIR, 'data')
    : app.getPath('userData')
  return path.join(base, 'license.json')
}

// ─── Validation cryptographique d'une clé ────────────────────────────────────
function verifyKey(licenseKey: string): { ok: boolean; payload?: any } {
  try {
    const parts = licenseKey.trim().split('-')
    if (parts.length < 3 || parts[0] !== 'MTC1') return { ok: false }
    const sig        = parts[parts.length - 1]
    const payloadB64 = parts.slice(1, -1).join('-')
    const expected   = crypto.createHmac('sha256', HMAC_SECRET)
                             .update(payloadB64)
                             .digest('hex')
                             .slice(0, 8)
    if (expected !== sig) return { ok: false }
    const payload = JSON.parse(Buffer.from(payloadB64, 'base64').toString('utf8'))
    return { ok: true, payload }
  } catch {
    return { ok: false }
  }
}

// ─── Vérification de l'expiration ────────────────────────────────────────────
function daysUntil(dateStr: string): number {
  if (dateStr === 'perpetual') return 9999
  const diff = new Date(dateStr).getTime() - Date.now()
  return Math.ceil(diff / 86400000)
}

// ─── Activation : enregistre la licence après vérification ───────────────────
export function activateLicense(licenseKey: string): LicenseInfo {
  const { ok, payload } = verifyKey(licenseKey)
  if (!ok || !payload) return { valid: false, mode: 'invalid' }

  const days = daysUntil(payload.expiry)
  if (days <= 0) return { valid: false, mode: 'expired', ...payload }

  fs.writeFileSync(getLicensePath(), JSON.stringify({
    key: licenseKey, activatedAt: new Date().toISOString()
  }), 'utf8')

  return { valid: true, mode: 'licensed', daysLeft: days, ...payload }
}

// ─── Lecture du statut au démarrage ──────────────────────────────────────────
export function getLicenseStatus(): LicenseInfo {
  const lp = getLicensePath()

  // 1. Licence activée ?
  if (fs.existsSync(lp)) {
    try {
      const { key } = JSON.parse(fs.readFileSync(lp, 'utf8'))
      const { ok, payload } = verifyKey(key)
      if (ok && payload) {
        const days = daysUntil(payload.expiry)
        if (days > 0) return { valid: true, mode: 'licensed', daysLeft: days, ...payload }
        return { valid: false, mode: 'expired', ...payload }
      }
    } catch { /* fichier corrompu → trial */ }
  }

  // 2. Mode trial : date du premier lancement
  const trialPath = lp.replace('license.json', 'trial.json')
  let trialStart: string
  if (fs.existsSync(trialPath)) {
    try { trialStart = JSON.parse(fs.readFileSync(trialPath, 'utf8')).start }
    catch { trialStart = new Date().toISOString() }
  } else {
    trialStart = new Date().toISOString()
    fs.writeFileSync(trialPath, JSON.stringify({ start: trialStart }), 'utf8')
  }

  const trialDaysLeft = TRIAL_DAYS - Math.floor(
    (Date.now() - new Date(trialStart).getTime()) / 86400000
  )

  if (trialDaysLeft > 0) {
    return { valid: true, mode: 'trial', trialDaysLeft }
  }

  return { valid: false, mode: 'expired', trialDaysLeft: 0 }
}

8. Code — Interface d'activation (Renderer)

src/renderer/pages/ActivationPage.tsx
import React, { useState } from 'react'

interface Props {
  mode: 'trial' | 'expired' | 'invalid'
  trialDaysLeft?: number
  onActivated: () => void
}

export function ActivationPage({ mode, trialDaysLeft, onActivated }: Props) {
  const [key, setKey]     = useState('')
  const [error, setError] = useState('')
  const [loading, setLoading] = useState(false)

  const handleActivate = async () => {
    if (!key.trim()) return
    setLoading(true); setError('')
    try {
      const result = await window.mtcApi.activateLicense(key.trim())
      if (result.valid) { onActivated() }
      else { setError('Clé invalide ou expirée. Vérifiez la clé fournie.') }
    } catch {
      setError('Erreur lors de la validation.')
    } finally { setLoading(false) }
  }

  return (
    <div style={{ display:'flex', alignItems:'center', justifyContent:'center',
      height:'100vh', background:'var(--bg)' }}>
      <div className="card" style={{ maxWidth:460, width:'100%', padding:36 }}>
        <h2 style={{ marginBottom:8, color:'var(--accent)' }}>Dossier Patient MTC</h2>

        {mode === 'trial' && trialDaysLeft !== undefined && (
          <div style={{ background:'#fff3cd', padding:'10px 14px', borderRadius:6,
            marginBottom:20, fontSize:13 }}>
            ⏳ Mode essai — {trialDaysLeft} jour{trialDaysLeft > 1 ? 's' : ''} restant{trialDaysLeft > 1 ? 's' : ''}
          </div>
        )}
        {mode === 'expired' && (
          <div style={{ background:'#fde8e8', padding:'10px 14px', borderRadius:6,
            marginBottom:20, fontSize:13, color:'#c0392b' }}>
            ⛔ Période d'essai terminée. Entrez votre clé de licence pour continuer.
          </div>
        )}

        <div className="field">
          <label>Clé de licence</label>
          <input
            type="text" className="input"
            placeholder="MTC1-xxxx-xxxx-xxxx"
            value={key}
            onChange={e => setKey(e.target.value)}
            onKeyDown={e => e.key === 'Enter' && handleActivate()}
          />
        </div>

        {error && <p style={{ color:'var(--red)', fontSize:12, marginBottom:10 }}>{error}</p>}

        <div style={{ display:'flex', gap:10, marginTop:8 }}>
          <button className="btn btn-primary" onClick={handleActivate} disabled={loading}>
            {loading ? 'Vérification…' : 'Activer'}
          </button>
          {mode === 'trial' && (
            <button className="btn btn-secondary" onClick={onActivated}>
              Continuer en mode essai
            </button>
          )}
        </div>
      </div>
    </div>
  )
}

  // Afficher une bannière discrète (optionnel)
}

12. Sécurité — Ce qui est protégé

RisqueProtection
Falsifier une cléImpossible sans la clé secrète HMAC (non distribuée)
Modifier la date d'expirationTout changement invalide la signature
Copier la clé sur un autre posteNon bloqué (offline) — acceptable pour v1
Extraire la clé secrèteCompilée dans le main process (non accessible du renderer)
Supprimer license.jsonRetour en mode trial (puis expiration)
Note : Pour une protection plus forte (verrouillage machine), une v2 peut ajouter l'identifiant matériel (fingerprint CPU/disque) dans le payload. Mais pour une v1 commerciale, ce niveau est largement suffisant.
