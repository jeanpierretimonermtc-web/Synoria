Installation

Installer Synoria étape par étape.

Guide complet pour installer Synoria sur Windows ou macOS, du téléchargement au premier lancement.

🪟 Windows Windows 10 / 11 — 64 bits 🍎 macOS Apple Silicon ou Intel

🪟Installation Windows

Prérequis

  • Windows 10 ou Windows 11, version 64 bits
  • Droits administrateur pour l'installation (une confirmation UAC sera demandée)
  • Environ 250 Mo d'espace disque

Étapes d'installation

1
Télécharger l'installateur

Rendez-vous sur logiciel-synoria.fr/telechargement et cliquez sur Windows — Installateur pour télécharger le fichier Synoria Setup 1.x.x.exe.

2
Lancer l'installateur

Double-cliquez sur le fichier téléchargé. Windows peut afficher un avertissement SmartScreen — cliquez sur "Informations complémentaires" puis "Exécuter quand même".

3
Confirmer l'élévation UAC

Une fenêtre demande l'autorisation d'administrateur. Cliquez sur Oui. L'installation se fait dans C:\Program Files\Synoria\.

4
Installation automatique

L'installateur s'exécute en quelques secondes. Un raccourci Synoria est créé sur le bureau et dans le menu Démarrer.

5
Premier lancement

Cliquez sur le raccourci Synoria. Définissez votre mot de passe de protection au premier écran, puis commencez à utiliser le logiciel.

Vos données sont stockées dans C:\Users\[votre nom]\AppData\Roaming\Synoria\. Ce dossier est indépendant du logiciel : une désinstallation ne l'efface pas.

Désinstaller Synoria

Depuis le menu Démarrer, faites un clic droit sur Synoria et choisissez Désinstaller, ou passez par Paramètres Windows → Applications → Synoria. Vos données patients ne sont pas supprimées automatiquement.

🍎Installation macOS

Prérequis

  • macOS 11 (Big Sur) ou version ultérieure
  • Apple Silicon (M1, M2, M3…) → téléchargez le fichier arm64.dmg
  • Processeur Intel → téléchargez le fichier x64.dmg
Pour vérifier votre puce : menu Pomme → À propos de ce Mac. Si vous voyez "Apple M…" → arm64. Si vous voyez "Intel Core…" → x64.

Identifier votre version

Cliquez sur le menu Pomme () en haut à gauche de l'écran, puis À propos de ce Mac. La puce est indiquée sous le nom du modèle.

Étapes d'installation

1
Télécharger le bon fichier .dmg

Sur logiciel-synoria.fr/telechargement, choisissez macOS Apple Silicon ou macOS Intel selon votre Mac.

2
Ouvrir le fichier .dmg

Double-cliquez sur le fichier téléchargé pour monter l'image disque.

3
Glisser Synoria dans Applications

Dans la fenêtre qui s'ouvre, faites glisser l'icône Synoria vers le dossier Applications.

4
Premier lancement : clic droit → Ouvrir

Ouvrez le dossier Applications dans le Finder. Faites un clic droit sur l'icône Synoria (ou Ctrl+clic), puis choisissez "Ouvrir" dans le menu contextuel. macOS affichera une boîte de dialogue — cliquez à nouveau sur "Ouvrir" pour confirmer. Cette étape n'est nécessaire qu'une seule fois.

5
Premier lancement

Définissez votre mot de passe de protection et commencez à utiliser Synoria.

Les données sont stockées dans ~/Library/Application Support/Synoria/.

Un problème lors de l'installation ?

Contactez le support — réponse sous 48h ouvrées.
