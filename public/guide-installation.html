Guide d'installation

Ce guide couvre l'installation de Synoria v1.5.8 sur Windows, la création de votre compte, la configuration du mot de passe de chiffrement, et le transfert vers un autre poste.

Téléchargement

  1. Rendez-vous sur synoria.fr et connectez-vous à votre espace personnel avec l'email utilisé lors de votre souscription.

  2. Dans la section Téléchargements, cliquez sur Installateur Windows .exe pour télécharger la version 1.5.8.

  3. Enregistrez le fichier dans votre dossier Téléchargements. Le nom du fichier ressemble à Synoria-Setup-1.5.8.exe.

Téléchargez uniquement depuis synoria.fr

Ne téléchargez jamais Synoria depuis un site tiers. Vérifiez le cadenas HTTPS dans la barre d'adresse de votre navigateur.

Installation Windows

  1. Double-cliquez sur le fichier Synoria-Setup-1.5.8.exe téléchargé.

  2. Windows SmartScreen peut afficher un avertissement "Windows a protégé votre ordinateur". Cliquez sur Informations complémentaires puis Exécuter quand même. Synoria est un logiciel légitime signé numériquement.

  3. Si Windows demande les droits administrateur (fenêtre bleue UAC), cliquez Oui.

  4. L'installeur NSIS s'exécute, copie les fichiers et crée un raccourci sur le bureau. L'installation dure moins d'une minute.

  5. Synoria se lance automatiquement à la fin de l'installation.

Contournement de SmartScreen (détail)

SmartScreen bloque les exécutables peu répandus à leur sortie. Ce n'est pas une indication de danger :

  • Cliquez Informations complémentaires (lien bleu en bas à gauche de la fenêtre)
  • Cliquez Exécuter quand même
Problème UAC uniquement

Si vous n'avez pas les droits administrateur sur votre ordinateur professionnel, demandez à votre responsable informatique d'effectuer l'installation.

Premier lancement

Au premier lancement, Synoria vous guide en 3 étapes :

Étape 1 — Connexion à votre compte Synoria

Synoria utilise un compte en ligne pour gérer votre abonnement. Vos données patients, elles, restent 100 % locales sur votre ordinateur.

  • Si vous avez déjà un compte : saisissez votre email et mot de passe Synoria, puis cliquez Se connecter.
  • Si vous n'avez pas encore de compte : cliquez Créer un compte, renseignez votre email professionnel et choisissez un mot de passe (8 caractères minimum). Acceptez les CGU, puis cliquez Créer mon compte.
Confirmez votre email

Après la création, vérifiez votre boîte mail et cliquez sur le lien de confirmation avant de vous connecter. Sans confirmation, la connexion est refusée. Si vous ne recevez pas l'email dans 5 minutes, vérifiez vos spams ou cliquez "Renvoyer l'email de confirmation" sur l'écran de connexion.

Étape 2 — Abonnement et essai gratuit

Si votre compte est nouveau ou si vous n'avez pas encore d'abonnement, l'écran d'abonnement s'affiche. Deux formules sont disponibles :

FormulePrixEssai gratuit
Synoria Annuel123 € / an (10,25 €/mois)14 jours gratuits
Synoria 6 mois63 € / 6 mois (10,50 €/mois)14 jours gratuits

Cliquez Commencer l'essai gratuit sur la formule de votre choix. Vous serez redirigé vers Stripe pour renseigner votre carte bancaire. Aucun prélèvement n'est effectué avant J+14.

Si vous avez déjà un abonnement actif

Cette étape est ignorée et Synoria passe directement à la création du mot de passe principal.

Étape 3 — Assistant de configuration

Au tout premier démarrage, un assistant (wizard) s'affiche une seule fois pour vous aider à choisir votre formulaire métier (MTC, kinésio, ostéo, naturo, basique, etc.) et configurer les sauvegardes automatiques.

Mot de passe principal — à lire absolument

Synoria utilise deux mots de passe distincts :

Mot de passeRôleOù le changer
Compte SynoriaConnexion à votre compte en ligne, gestion de l'abonnementSur synoria.fr
Mot de passe principalChiffre votre base de données patients (AES-256) ET chiffre vos sauvegardesParamètres → Sécurité
Ce mot de passe est irremplaçable

Il n'existe aucune procédure de récupération du mot de passe principal. Si vous le perdez, vous perdez l'accès à votre base de données ET à vos sauvegardes chiffrées. Notez-le immédiatement dans un gestionnaire de mots de passe ou en lieu sûr physique (coffre, tiroir fermé à clé).

Créer le mot de passe principal

  1. L'écran de création s'affiche au premier lancement. Saisissez un mot de passe d'au moins 6 caractères (recommandé : 12+ caractères avec des chiffres et caractères spéciaux).

  2. Confirmez-le dans le second champ.

  3. Cliquez Créer le mot de passe. Synoria chiffre immédiatement la base de données et ouvre l'application.

Déverrouillage au démarrage

À chaque démarrage suivant, Synoria demande le mot de passe principal pour déchiffrer la base de données. L'application se verrouille également automatiquement après 20 minutes d'inactivité ou via Ctrl+L.

Transfert vers un autre ordinateur

Pour installer Synoria sur un autre ordinateur et retrouver toutes vos données, suivez ces étapes dans l'ordre.

Sur l'ancien ordinateur — Créer une sauvegarde transférable

  1. Dans Synoria, cliquez sur le bouton Données dans l'en-tête (en haut à droite), puis Sauvegarder la base de données. Ou allez dans Paramètres → Sauvegardes → Sauvegarder maintenant.

  2. Un fichier backup-global-AAAA-MM-JJ-HHhMM.json.enc est créé dans votre dossier de sauvegardes. Copiez ce fichier sur une clé USB ou un dossier partagé (cloud personnel, email, etc.).

  3. Notez votre mot de passe principal — il sera nécessaire pour restaurer la sauvegarde sur le nouvel ordinateur.

Sauvegardes chiffrées par votre mot de passe

Les sauvegardes Synoria sont chiffrées avec votre mot de passe principal. Elles sont directement importables sur n'importe quel ordinateur — il suffit de connaître votre mot de passe.

Sur le nouvel ordinateur — Installer et restaurer

  1. Installez Synoria en suivant les étapes de ce guide (téléchargement → installation → connexion au compte Synoria → abonnement).

  2. Lors de la création du mot de passe principal, choisissez le même mot de passe que sur l'ancien ordinateur. Cela n'est pas obligatoire, mais facilite la mémorisation.

  3. Une fois dans l'application, cliquez Données → Restaurer une sauvegarde… (ou Paramètres → Sauvegardes → Importer une sauvegarde).

  4. Sélectionnez le fichier .json.enc copié depuis l'ancien ordinateur.

  5. Une fenêtre s'ouvre — saisissez le mot de passe principal de l'ancien ordinateur et cliquez Importer. Synoria fusionne les données et affiche un résumé (patients importés, séances importées, doublons ignorés).

Désactiver l'ancien appareil

Votre licence autorise 2 appareils simultanément. Si l'ancien ordinateur est remplacé définitivement, désactivez-le pour libérer un slot :

  • Sur le nouvel ordinateur : Mon abonnement → Appareils actifs → trouvez l'ancien poste → Désactiver.
  • Ou sur l'ancien ordinateur (s'il est encore accessible) : Mon abonnement → Désactiver cet appareil.

Mises à jour

Synoria vérifie automatiquement les mises à jour. Lorsqu'une nouvelle version est disponible, un bandeau bleu apparaît sous la barre de navigation.

Procédure de mise à jour

  1. Cliquez sur Télécharger vX.Y.Z dans le bandeau de mise à jour (ou via Mon abonnement → Télécharger). Le navigateur télécharge le nouveau fichier .exe.

  2. Une fois le téléchargement terminé, allez dans Paramètres → Sécurité → Mise à jour du logiciel. Cliquez sur le bouton Parcourir pour sélectionner le fichier .exe téléchargé.

  3. Cliquez Installer la mise à jour. Synoria se ferme automatiquement et lance l'installeur.

  4. Suivez l'installeur, puis relancez Synoria depuis le bureau.

Vos données ne sont pas supprimées

Une mise à jour ne touche jamais vos données patients. La base de données chiffrée reste intacte dans AppData\Roaming\Synoria\.

Mises à jour requises

Certaines versions sont marquées "requises" (⚠). Dans ce cas, le bandeau ne peut pas être ignoré et le bouton "Télécharger" est mis en évidence. Il est recommandé d'effectuer ces mises à jour dès que possible.

Désinstallation

Sauvegardez avant de désinstaller

La désinstallation ne supprime pas vos données (elles sont dans AppData), mais il est prudent de faire une sauvegarde complète avant toute manipulation.

  1. Faites une sauvegarde complète : Données → Sauvegarder la base de données.

  2. Fermez Synoria complètement.

  3. Dans Windows : Paramètres → Applications → Applications installées → recherchez "Synoria" → Désinstaller.

  4. Pour supprimer aussi vos données patients (attention, irréversible) : supprimez manuellement le dossier C:\Users\[nom]\AppData\Roaming\Synoria\.

Foire aux questions

Je n'arrive pas à me connecter — "Email non confirmé"

Vérifiez votre boîte mail (y compris les spams). Cliquez sur le lien dans l'email de confirmation. Si vous ne trouvez pas l'email, sur l'écran de connexion Synoria, cliquez Renvoyer l'email de confirmation.

J'ai oublié mon mot de passe Synoria (compte)

Sur l'écran de connexion, cliquez Mot de passe oublié ?. Saisissez votre email et cliquez Envoyer. Vous recevrez un lien de réinitialisation par email.

J'ai oublié mon mot de passe PRINCIPAL (base de données)

Ce mot de passe est irrécupérable

Le mot de passe principal chiffre votre base de données avec AES-256. Il n'est jamais envoyé à Synoria. Si vous le perdez, il n'existe aucune procédure de récupération. Notez-le impérativement.

Puis-je installer Synoria sur plusieurs ordinateurs ?

Oui, votre licence autorise jusqu'à 2 appareils simultanés. La synchronisation automatique entre postes n'est pas disponible — utilisez les sauvegardes pour transférer vos données d'une machine à l'autre. Pour ajouter un 3ème appareil, désactivez d'abord l'un des deux actifs depuis Mon abonnement → Appareils actifs.

Où sont stockées mes données ?

FichierEmplacement Windows
Base de données (chiffrée)%AppData%\Synoria\database\mtc.sqlite.enc
Paramètres%AppData%\Synoria\settings.json
Formulaire actif%AppData%\Synoria\active.plugin.json

Le dossier AppData est caché par défaut dans Windows. Pour l'afficher : Explorateur de fichiers → Affichage → Éléments masqués. Ou appuyez sur Win+R et saisissez %AppData%\Synoria.

L'installation échoue / l'installeur ne se lance pas

Faites un clic droit sur le fichier .exe → Exécuter en tant qu'administrateur. Si le problème persiste, vérifiez que votre Windows est à jour (Windows Update) et redémarrez.

Synoria ne démarre pas après l'installation

Redémarrez votre ordinateur et relancez Synoria. Si le problème persiste, contactez le support (contact@logiciel-synoria.fr) en joignant le rapport de diagnostic (Paramètres → Support → Générer le rapport de diagnostic).

Ma sauvegarde ne peut pas être restaurée sur le nouvel ordinateur

Vérifiez que vous utilisez le bon mot de passe principal (celui de l'ancien ordinateur). Vous pouvez aussi contrôler l'intégrité du fichier avant de l'importer : Paramètres → Sauvegardes → Vérifier une sauvegarde.
