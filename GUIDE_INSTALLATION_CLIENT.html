Étape 2
Installation sur PC (version installée)
✅ Recommandé pour une utilisation fixe — le logiciel s'installe sur votre ordinateur comme n'importe quel programme Windows.
  1. 1
    Ouvrez le dossier contenant le fichier d'installation
    Trouvez le fichier Dossier Patient MTC Setup 1.2.0.exe dans vos téléchargements ou sur la clé USB reçue.
  2. 2
    Double-cliquez sur le fichier pour lancer l'installation
    Windows peut afficher un avertissement de sécurité en bleu ou orange. C'est normal pour les logiciels non signés.
  3. 3
    Si Windows affiche "Windows a protégé votre PC"
    Cliquez sur "Informations complémentaires" (lien bleu en bas à gauche), puis sur le bouton "Exécuter quand même" qui apparaît.
  4. 4
    Suivez les étapes de l'installateur
    Cliquez sur "Suivant" puis "Installer". L'installation dure environ 30 secondes. Le logiciel s'ouvre automatiquement à la fin.
⚠️ Antivirus : certains antivirus peuvent bloquer l'installation. Si c'est le cas, ajoutez une exception pour ce fichier, ou désactivez temporairement votre antivirus pendant l'installation (puis réactivez-le ensuite).

Étape 3
Création de votre mot de passe

Au premier lancement, le logiciel vous demande de créer un mot de passe. Ce mot de passe protège l'accès au logiciel et chiffre vos données patients (cryptage AES-256).

🔐 Créer le mot de passe & sécuriser
  1. 1
    Choisissez un mot de passe d'au moins 6 caractères
    Exemple : Cabinet2026! ou MonNomDate. Évitez les mots trop simples comme 1234 ou password.
  2. 2
    Confirmez le mot de passe dans le second champ
    Les deux champs doivent être identiques. Un message d'erreur apparaît s'ils sont différents.
  3. 3
    Cliquez sur "Créer le mot de passe & sécuriser"
    Le chiffrement des données se fait automatiquement (quelques secondes). Le logiciel s'ouvre ensuite normalement.
⛔ Important : Notez votre mot de passe sur un support physique (carnet, Post-it dans un endroit sûr). Sans lui, vos données seront irrécupérables. Il n'existe pas de procédure de récupération.

Étape 4
Importer votre plugin de spécialité

Le plugin est un fichier .json qui personnalise le formulaire de séance selon votre spécialité. Sans plugin, le logiciel utilise le formulaire MTC intégré.

📂 Prérequis : Vous devez avoir reçu votre fichier plugin (exemple : Kinesio_Charlotte.plugin.json ou MTC_JP.plugin.json). Enregistrez-le sur votre ordinateur dans un endroit facile à retrouver.

Accéder aux Paramètres

  1. 1
    Ouvrez le logiciel et connectez-vous avec votre mot de passe
    Le tableau de bord s'affiche après la connexion.
  2. 2
    Cliquez sur ⚙️ "Paramètres" dans la barre de navigation en haut
    La barre de navigation est en haut du logiciel. Cliquez sur l'icône ⚙️ tout à droite.
  3. 3
    Faites défiler jusqu'à la section "Plugin de spécialité"
    Cette section se trouve au milieu de la page Paramètres, après les sauvegardes.
⚙️ Paramètres — Plugin de spécialité
🌿
Formulaire MTC intégré
Acupuncture · Barrage · Diagnostic MTC · Traitement · Systèmes énergétiques
Importez un fichier .json de plugin pour remplacer le formulaire MTC par l'anamnèse de votre spécialité.
📥 Importer un plugin (.json)
  1. 4
    Cliquez sur "📥 Importer un plugin (.json)"
    Une fenêtre de sélection de fichier s'ouvre.
  2. 5
    Naviguez jusqu'à votre fichier plugin et sélectionnez-le
    Exemple : Kinesio_Charlotte.plugin.json ou MTC_JP.plugin.json. Double-cliquez dessus ou sélectionnez-le et cliquez "Ouvrir".
  3. 6
    Un message de confirmation apparaît en haut de l'écran
    Exemple : "Plugin Kinésiologie v1.0.0 installé ✓". La section affiche maintenant le nom de votre spécialité.
✅ C'est tout ! Dès maintenant, la page "Nouvelle séance" affiche le formulaire de votre spécialité à la place du formulaire MTC. Allez tester en cliquant sur "Nouvelle séance" dans la navigation.

Version clé USB (sans installation)
💡 Idéal si vous travaillez sur plusieurs ordinateurs ou si vous préférez ne rien installer. Le logiciel et vos données sont entièrement sur la clé USB.
  1. 1
    Copiez le dossier fourni sur votre clé USB
    Le dossier s'appelle CleUSB_DossierPatientMTC. Copiez-le à la racine de la clé USB (pas dans un sous-dossier). Contenu du dossier :
    ⚙️
    Dossier Patient MTC.exe
    Le logiciel — double-cliquer pour lancer
    📁
    data\
    Vos données patients — ne jamais supprimer
    📄
    LIRE_MOI.txt
    Instructions rapides
  2. 2
    Branchez la clé USB sur n'importe quel PC Windows
    Windows 10 ou 11, 64 bits. Aucune installation requise sur le PC.
  3. 3
    Ouvrez le dossier CleUSB_DossierPatientMTC et double-cliquez sur "Dossier Patient MTC.exe"
    Si Windows affiche un avertissement, cliquez "Informations complémentaires" puis "Exécuter quand même".
  4. 4
    Créez votre mot de passe (premier lancement uniquement)
    Même procédure que page 2 de ce guide. Ce mot de passe est lié à cette clé USB.
  5. 5
    Importez votre plugin (même procédure qu'à l'étape 4)
    Copiez d'abord votre fichier .plugin.json sur la clé USB, puis importez-le depuis Paramètres → Plugin de spécialité.
⚠️ Sauvegardez régulièrement ! Si vous perdez la clé USB, vous perdez vos données. Utilisez le bouton "💾 Sauvegarder" dans l'application pour copier une sauvegarde sur votre PC ou un autre support.

Étape 5
Mise à jour du logiciel

Quand une nouvelle version du logiciel est disponible, vous recevez un fichier .exe par email. Voici comment mettre à jour sans perdre vos données.

  1. 1
    Enregistrez le nouveau fichier .exe sur votre PC
    Exemple : Dossier Patient MTC Setup 1.3.0.exe. Notez bien l'endroit où vous l'enregistrez.
  2. 2
    Ouvrez le logiciel actuel → ⚙️ Paramètres
    Faites défiler jusqu'à la section "Mise à jour de l'application".
  3. 3
    Cliquez "📥 Sélectionner le fichier de mise à jour (.exe)"
    Naviguez jusqu'au nouveau fichier .exe et sélectionnez-le.
  4. 4
    Cliquez "🚀 Installer la mise à jour"
    Le logiciel se ferme automatiquement et l'installation démarre. Vos données et votre mot de passe sont conservés.
  5. 5
    Relancez le logiciel et reconnectez-vous avec votre mot de passe
    Votre plugin de spécialité est toujours actif, vos données sont intactes.
🔌 Mise à jour du plugin : Si votre formulaire de spécialité évolue, vous pouvez recevoir un nouveau fichier plugin. Importez-le simplement dans Paramètres → Plugin de spécialité → "Remplacer le plugin". Vos données existantes ne sont pas affectées.

Aide
Questions fréquentes
❓ Windows dit que le fichier est dangereux — que faire ?
C'est normal pour les logiciels non signés numériquement. Cliquez "Informations complémentaires" puis "Exécuter quand même". Le logiciel est sûr et ne contient aucun virus.
❓ J'ai oublié mon mot de passe — comment récupérer mes données ?
Malheureusement, le mot de passe est la seule clé de chiffrement de vos données. Sans lui, les données sont irrécupérables. C'est pourquoi il est essentiel de le noter. Contactez le support pour de l'aide : jeanpierre.timoner.mtc@gmail.com
❓ Mon plugin ne s'est pas installé — un message d'erreur est apparu
Vérifiez que vous sélectionnez bien un fichier .json et non .exe ou autre. Si le problème persiste, envoyez le message d'erreur exact au support.
❓ J'ai installé le plugin mais je vois toujours l'ancien formulaire
Fermez et relancez le logiciel. Si le problème persiste, allez dans Paramètres → Plugin et vérifiez que votre plugin est bien affiché avec son nom.
❓ L'application s'ouvre sur un écran blanc
Fermez complètement l'application (croix en haut à droite) et relancez-la. Si le problème persiste, redémarrez votre ordinateur.
❓ Mes données sont-elles sauvegardées automatiquement ?
Oui — le logiciel sauvegarde chiffré toutes les 3 minutes pendant la session, et automatiquement à la fermeture si vous avez activé l'option dans Paramètres → Automatisation. Vous pouvez aussi lancer une sauvegarde manuelle avec le bouton "💾 Sauvegarder".
❓ Puis-je utiliser le logiciel sur plusieurs ordinateurs ?
Oui, avec la version clé USB : branchez la clé sur n'importe quel PC Windows et lancez le logiciel directement. Vos données suivent la clé. Pour synchroniser entre deux PC fixes, utilisez les sauvegardes manuelles (exporter d'un PC, importer sur l'autre).
