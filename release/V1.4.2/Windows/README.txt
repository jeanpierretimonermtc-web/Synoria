SYNORIA v1.4.2 — Version Windows
==================================

FICHIER D'INSTALLATION
-----------------------
  Synoria Setup 1.4.2.exe   — Installateur NSIS (Windows 10/11 64 bits)

INSTALLATION
------------
1. Double-cliquez sur "Synoria Setup 1.4.2.exe"
2. Suivez les étapes de l'assistant d'installation
3. Synoria est installé dans Program Files et ajouté au menu Démarrer
4. Un raccourci est créé sur le bureau

MISE À JOUR depuis v1.4.1
--------------------------
L'installateur détecte automatiquement l'ancienne version et la remplace.
Vos données, paramètres et mot de passe sont intégralement conservés.

DÉSINSTALLATION
---------------
Paramètres Windows → Applications → Synoria → Désinstaller
(Vos données ne sont PAS supprimées lors de la désinstallation)
