Installation Windows

Fichier à utiliser

📦
Synoria Setup 1.3.1.exe
Taille : environ 83 Mo · Installateur NSIS
Installe Synoria dans C:\Program Files\Synoria\ et crée un raccourci sur le bureau

Procédure d'installation

1
Double-cliquez sur Synoria Setup 1.3.1.exe
Si le fichier est dans Téléchargements, ouvrez l'Explorateur → Téléchargements → double-cliquez
2
Windows affiche une alerte de sécurité — c'est normal
Vous verrez probablement l'un de ces écrans :
⚠️ SmartScreen Windows
Cliquez "Informations complémentaires"
puis "Exécuter quand même"
⚠️ Contrôle de compte
Cliquez "Oui" pour autoriser l'installation
Ces alertes sont normales pour tout logiciel indépendant non signé numériquement. Synoria ne contient aucun code malveillant.
3
L'installation se lance automatiquement
Une barre de progression s'affiche. Durée : 30 à 60 secondes selon votre PC.
4
Installation terminée
Un raccourci Synoria apparaît sur le bureau. L'application peut se lancer automatiquement.

Ce qui est installé

ÉlémentEmplacement
ApplicationC:\Program Files\Synoria\
Raccourci bureauBureau\Synoria.lnk
Raccourci menu démarrerMenu Démarrer\Synoria\
Données patients (chiffrées)%APPDATA%\Synoria\
DésinstallateurC:\Program Files\Synoria\Uninstall Synoria.exe
⚠️ Important : L'application s'installe dans Program Files mais vos données patients sont stockées séparément dans %APPDATA%\Synoria\. La désinstallation de l'application ne supprime pas vos données.
3
Version clé USB (sans installation)
✅ Avantage clé USB : Aucune installation requise. Branchez la clé sur n'importe quel PC Windows et lancez directement. Vos données voyagent avec vous.

Fichier à utiliser

📄
Synoria 1.3.1.exe
Taille : environ 83 Mo · Exécutable portable (pas d'installation)

Préparer la clé USB

1
Créez un dossier sur la clé USB
Exemple : Synoria\ à la racine de la clé USB
2
Copiez les fichiers dans ce dossier
Synoria\ ├── Synoria 1.3.1.exe ← l'application ├── LIRE_MOI.txt ← instructions rapides └── Guide_Installation_…html ← ce guide
3
Premier lancement
Double-cliquez sur Synoria 1.3.1.exe. Un dossier data\ est automatiquement créé à côté de l'exe pour stocker vos données.

Structure finale sur la clé USB

Synoria\ ├── Synoria 1.3.1.exe ← application portable ├── data\ ← créé automatiquement au 1er lancement │ ├── database\mtc.sqlite.enc ← votre base chiffrée │ ├── auth.json ← vérificateur mot de passe │ ├── encryption.key ← clé sauvegardes │ └── settings.json ← vos paramètres ├── LIRE_MOI.txt └── Guide_Installation_….html
🔴 Ne jamais supprimer le dossier data\ — il contient votre base de données chiffrée. Sans lui, toutes vos données seront perdues.

Utilisation sur un autre PC

1
Branchez la clé USB sur le PC
2
Ouvrez l'explorateur → votre clé USB → dossier Synoria
3
Double-cliquez sur Synoria 1.3.1.exe → vos données se chargent automatiquement

Installation macOS

Choisir la bonne version

💻 Mac Intel

Fichier : Synoria-1.3.1-x64.dmg

Pour les Mac fabriqués avant fin 2020 (processeur Intel Core i5/i7/i9)

Comment vérifier : Menu Pomme → À propos de ce Mac → si "Intel Core" apparaît

⚡ Mac Apple Silicon

Fichier : Synoria-1.3.1-arm64.dmg

Pour les Mac M1, M2, M3 (depuis fin 2020)

Comment vérifier : Menu Pomme → À propos de ce Mac → si "Apple M1/M2/M3" apparaît

Procédure d'installation

1
Double-cliquez sur le fichier .dmg
Une fenêtre s'ouvre avec l'icône Synoria et le dossier Applications
2
Glissez l'icône Synoria vers le dossier Applications
🌿 Synoria.app    →→→    📁 Applications
3
Éjectez le DMG (clic droit → Éjecter)
4
Ouvrez Synoria depuis Applications ou Launchpad

⚠️ Alerte Gatekeeper (première ouverture)

macOS peut bloquer l'application car elle n'est pas signée par un certificat Apple Developer payant :

macOS affiche : "Synoria ne peut pas être ouvert car son développeur ne peut pas être vérifié"
1
Fermez cette alerte
2
Clic droit sur l'icône Synoria → "Ouvrir"
3
Une nouvelle alerte apparaît avec le bouton "Ouvrir" → cliquez-le
Cette étape n'est nécessaire qu'une seule fois.
Alternative (macOS 13+) : Préférences Système → Confidentialité et sécurité → section "Sécurité" → cliquez "Ouvrir quand même" à côté de Synoria.

Premier lancement

Écran de création du mot de passe

Au tout premier démarrage, Synoria affiche l'écran de configuration du mot de passe :

🌿 SYNORIA
Créer un mot de passe
Choisissez un mot de passe fort pour chiffrer vos données patients
🔒 Ce mot de passe chiffre votre base en AES-256.
Sans lui, vos données seront illisibles. Notez-le en lieu sûr.
Nouveau mot de passe
Confirmer le mot de passe
🔐 Créer le mot de passe
1
Choisissez un mot de passe — minimum 6 caractères
Recommandé : au moins 10 caractères, avec chiffres et caractères spéciaux
2
Confirmez le mot de passe en le saisissant une deuxième fois
3
Cliquez "🔐 Créer le mot de passe"
La base de données est chiffrée. L'application s'ouvre automatiquement.
🔴 CRITIQUE — Notez votre mot de passe dès maintenant
Ce mot de passe est le seul moyen de déchiffrer vos données. Si vous l'oubliez, les données sont définitivement irrécupérables. Notez-le dans un gestionnaire de mots de passe (Bitwarden, 1Password) ou un endroit physiquement sécurisé.

Démarrage normal (connexions suivantes)

À chaque démarrage, Synoria affiche l'écran de connexion. Saisissez votre mot de passe et cliquez "🔓 Déverrouiller".

⏱️ Verrouillage automatique : Après 20 minutes d'inactivité, Synoria se verrouille automatiquement. Votre mot de passe est demandé pour reprendre.

Migration depuis une version précédente

Migration automatique (Windows)

Si vous avez déjà installé une version précédente ("Dossier Patient MTC"), Synoria v1.3.1 détecte et déplace automatiquement vos données au premier démarrage :

Avant : %APPDATA%\Dossier Patient MTC\ ← ancien dossier Après : %APPDATA%\Synoria\ ← nouveau dossier (déplacé automatiquement)
✅ Aucune action requise. Mot de passe, patients, séances, paramètres — tout est conservé. La migration se fait en une fraction de seconde au démarrage.

Migration manuelle (si la migration automatique n'a pas fonctionné)

1
Ouvrez l'Explorateur Windows → barre d'adresse → tapez %APPDATA% → Entrée
2
Vous devez voir un dossier Dossier Patient MTC
Si le dossier Synoria n'existe pas encore, renommez-le en Synoria
3
Relancez Synoria — vos données sont chargées automatiquement

Migration via sauvegarde (recommandé pour changer de PC)

1
Sur l'ancien PC/ancienne version : cliquez "💾 Sauvegarder" (bouton en haut à droite)
2
Copiez le fichier .enc généré sur une clé USB ou dans le cloud
3
Sur le nouveau PC : installez Synoria v1.3.1, créez un mot de passe
4
Cliquez "📂 Importer" → sélectionnez votre fichier .enc → toutes vos données sont restaurées
⚠️ Note : La sauvegarde utilise une clé de chiffrement distincte du mot de passe (fichier encryption.key dans %APPDATA%\Synoria\). Pour une migration complète d'un PC à l'autre, copiez aussi ce fichier si vous souhaitez réutiliser les anciennes sauvegardes.

Désinstallation

Windows

Méthode 1 — Via Windows :

  1. Paramètres Windows → Applications → "Synoria"
  2. Cliquez "Désinstaller"

Méthode 2 — Direct :

  1. Ouvrez C:\Program Files\Synoria\
  2. Double-cliquez Uninstall Synoria.exe
Vos données ne sont pas supprimées. Elles restent dans %APPDATA%\Synoria\. Supprimez ce dossier manuellement si souhaité.

macOS

  1. Ouvrez le dossier Applications
  2. Faites glisser Synoria.app vers la Corbeille
  3. Videz la Corbeille
Données Mac : stockées dans ~/Library/Application Support/Synoria/. Supprimez ce dossier si vous souhaitez effacer toutes les données.

Problèmes fréquents & solutions

🖥️ Windows

❌ "Windows a protégé votre PC" / SmartScreen bloque l'installation

Cause : Windows SmartScreen bloque les exécutables sans signature numérique EV (coût : 300-500€/an).

Solution : Cliquez "Informations complémentaires" → "Exécuter quand même". Cette alerte n'est affichée qu'à la première exécution.

❌ L'antivirus supprime ou bloque le fichier

Cause : Certains antivirus signalent les exécutables portables non signés comme suspects (faux positif).

Solution : Ajoutez le fichier ou le dossier en exception dans votre antivirus :
Windows Defender : Sécurité Windows → Protection contre les virus → Exclusions → Ajouter une exclusion → Fichier → sélectionnez Synoria Setup 1.3.1.exe

❌ Écran blanc ou l'application ne démarre pas

Solution 1 : Fermez complètement et relancez (parfois au premier démarrage)

Solution 2 : Clic droit sur le raccourci → "Exécuter en tant qu'administrateur"

Solution 3 : Vérifiez que Windows est à jour (Windows Update)

❌ Mot de passe refusé après migration

Cause : Le dossier de données n'a pas été migré correctement.

Solution : Vérifiez que %APPDATA%\Synoria\ contient bien les fichiers auth.json et database\mtc.sqlite.enc. Si le dossier est vide, copiez manuellement le contenu de %APPDATA%\Dossier Patient MTC\.

🍎 macOS

❌ "Synoria ne peut pas être ouvert" (Gatekeeper)

Solution : Clic droit sur Synoria.app → "Ouvrir" → dans la nouvelle boîte de dialogue → "Ouvrir". Cette étape n'est nécessaire qu'une seule fois.

❌ "L'application est endommagée" (macOS Ventura +)

Solution : Ouvrez Terminal et tapez :

$ xattr -cr /Applications/Synoria.app

Puis relancez l'application normalement.

❌ Mauvaise version DMG (Intel vs Apple Silicon)

Vérification : Menu Pomme → À propos de ce Mac → regardez le champ "Processeur"

  • Affiche "Intel Core …" → utilisez Synoria-1.3.1-x64.dmg
  • Affiche "Apple M1/M2/M3" → utilisez Synoria-1.3.1-arm64.dmg
