1. Installation & premiers pas

Installation standard (PC fixe ou portable)

1
Double-cliquez sur Synoria Setup 1.3.1.exe
2
Si Windows affiche "Application inconnue", cliquez "Informations complémentaires" → "Exécuter quand même" (comportement normal pour logiciel non signé)
3
Suivez l'assistant d'installation — l'application s'installe dans C:\Program Files\Synoria\
4
Un raccourci est créé sur le bureau. Double-cliquez dessus pour lancer Synoria.
⚠️ Antivirus : certains antivirus peuvent signaler le fichier comme "suspect". C'est un faux positif courant pour les logiciels indépendants non signés numériquement. Ajoutez une exception dans votre antivirus si nécessaire.

Version clé USB (sans installation)

Voir la section 14 — Version clé USB.

2. Sécurité & mot de passe

Premier démarrage — Création du mot de passe

Au premier lancement, Synoria vous demande de créer un mot de passe. Ce mot de passe chiffre l'intégralité de votre base de données en AES-256-GCM.

🔴 IMPORTANT : Sans ce mot de passe, vos données sont totalement irrécupérables. Notez-le dans un endroit sûr (coffre-fort numérique, carnet sécurisé…).
AspectDétail
AlgorithmeAES-256-GCM (standard bancaire)
Dérivation de cléPBKDF2 — 600 000 itérations, SHA-256
Longueur minimale6 caractères
Verrouillage auto20 minutes d'inactivité
Sauvegarde automatique chiffréeToutes les 3 minutes pendant la session

Changer le mot de passe

Allez dans ⚙️ Paramètres → onglet 🔐 Sécurité → section "Changer le mot de passe".

4. Gestion des patients

Créer un nouveau patient

1
Cliquez sur 👤 Patients
2
Cliquez sur le bouton "+ Nouveau patient"
3
Renseignez au minimum le nom et le prénom
4
Cliquez "Enregistrer"

Informations disponibles sur la fiche patient

  • Identité : nom, prénom, date de naissance, profession
  • Contact : téléphone, email, adresse
  • Médical : médecin traitant, médicaments, antécédents
  • Notes générales & alertes importantes
  • Consentement RGPD (date et statut)

Archiver un patient

Ouvrez la fiche patient → bouton "Archiver". Le patient disparaît de la liste principale mais ses données sont conservées. Cochez "Afficher les archivés" pour les retrouver.

5. Créer et gérer une séance

Créer une nouvelle séance

1
Cliquez sur ➕ Nouvelle séance
2
Sélectionnez le patient dans la liste déroulante
3
Vérifiez la date (pré-remplie avec aujourd'hui)
4
Remplissez les sections pertinentes (motif, évolution, traitement…)
5
Cliquez "💾 Enregistrer la séance"

Sections du formulaire selon le mode

Mode Simple (sans plugin)

Motif · Évolution · Anamnèse · Traitement · Réactions · Plan de suivi

Mode MTC intégré (plugin MTC JP)

Motif · Évolution · Interrogatoire · Questionnaire systèmes · Observation MTC · Tests énergétiques · Analyse clinique · Traitement · Barrage homéopathique · Plan de suivi

Mode Plugin tiers (Kinésiologie, etc.)

Les sections sont définies par le fichier plugin JSON chargé dans Paramètres.

Brouillon automatique

Le formulaire sauvegarde automatiquement un brouillon toutes les 60 secondes. Si vous fermez l'application sans enregistrer, le brouillon sera proposé au prochain démarrage.

Prochain rendez-vous

En bas du formulaire, la section "Plan de suivi" permet de planifier le prochain RDV. Il sera automatiquement créé dans le Calendrier.

6. Calendrier & rendez-vous

Le calendrier offre deux vues :

  • Vue mensuelle (gauche) : aperçu du mois
  • Planning horaire (droite) : créneaux de 08h00 à 19h30 par pas de 30 min

Créer un rendez-vous

Cliquez sur un créneau dans le planning ou sur le bouton "+ RDV". Vous pouvez associer un patient existant ou renseigner les coordonnées d'un invité (nom, prénom, téléphone).

Couleurs des rendez-vous

CouleurSignification
🔵 BleuRDV planifié
🟢 VertSéance réalisée
🟠 OrangeEn attente / non confirmé

Depuis un RDV → créer la séance

Cliquez sur un RDV → bouton "Créer la séance". Vous êtes redirigé vers le formulaire Nouvelle séance avec le patient et la date pré-remplis.

7. Comptabilité & dépenses

Tableau de recettes

La page 📊 Comptabilité affiche un tableau mensuel avec :

  • Nombre de séances par type de consultation
  • Chiffre d'affaires mensuel calculé automatiquement
  • Taux URSAF paramétrable par mois
  • Cotisations URSAF estimées
  • Dépenses variables (publicité, logiciel, DASRI…)
  • Total annuel et export Excel

Configurer les types de consultation et tarifs

Charges fixes

Section "Charges fixes mensuelles" dans 📉 Dépenses :

  • Mois actifs configurables (charge saisonnière)

8. Factures

Générer une facture

1
Ouvrez le résumé d'une séance dans 📋 Séances
2
Cliquez sur "🧾 Générer la facture"
3
Vérifiez les informations et confirmez
4
Le PDF est généré et enregistré dans votre dossier de factures

Paramétrer la facturation

Dans ⚙️ Paramètres → onglet 🧾 Facturation :

  • Chemin du dossier de sauvegarde des factures
  • Taux de TVA (0% par défaut pour les soins)
  • Numérotation (année + numéro séquentiel)
  • Informations du praticien (nom, adresse, SIRET…)

9. Sauvegardes & restauration

Sauvegarde manuelle

Cliquez sur le bouton "💾 Sauvegarder" en haut à droite. La sauvegarde chiffrée (format .enc) est copiée dans deux dossiers configurables.

Sauvegarde automatique

Dans ⚙️ Paramètres → 💾 Sauvegardes :

  • À la fermeture : sauvegarde automatique quand vous quittez l'application
  • Quotidienne : une sauvegarde par jour au premier démarrage
  • Rétention : nombre de jours de conservation des sauvegardes anciennes

Restaurer une sauvegarde

1
Cliquez sur "📂 Importer" en haut à droite
2
Sélectionnez votre fichier de sauvegarde (.enc ou .json)
3
L'application fusionne les données (aucun doublon créé)
⚠️ Les sauvegardes utilisent une clé distincte du mot de passe. En cas de réinstallation sur un nouveau PC, copiez aussi le fichier encryption.key depuis l'ancien dossier de données.

10. RGPD & conformité

Synoria intègre les outils nécessaires à la conformité RGPD pour les praticiens de santé (Art. 9(2)(h)).

Consentements patients

Sur chaque fiche patient, cochez la case "Consentement donné" et enregistrez la date. Suivez le taux de consentement dans 🔒 RGPD → onglet "Consentements".

Journal d'accès

Chaque consultation de dossier est automatiquement journalisée (date, heure, action). Consultez-le dans 🔒 RGPD → onglet "Journal".

Alertes de conservation

L'application vous alerte quand des dossiers approchent ou dépassent la durée de conservation configurée (défaut : 10 ans). Paramétrez dans ⚙️ Paramètres → 🔒 RGPD.

Registre Art. 30

Générez le registre de traitement (obligatoire) depuis 🔒 RGPD → bouton "Exporter le registre". Le fichier HTML est enregistré dans votre dossier d'exports.

11. Synchronisation Google Calendar

Synoria peut synchroniser vos rendez-vous avec Google Calendar (visible sur votre téléphone Android ou iPhone).

🔒 Confidentialité : Seul le titre "Consultation" et l'horaire sont envoyés à Google. Aucun nom de patient ou donnée médicale ne quitte l'application.

Configuration (une seule fois)

1
Créez un projet sur console.cloud.google.com
2
Activez l'API Google Calendar API
3
Créez des identifiants OAuth 2.0 → "Application de bureau"
4
Ajoutez l'URI de redirection : http://127.0.0.1:42813/oauth2callback
5
Dans Synoria : ⚙️ Paramètres → 📅 Google Cal. → entrez Client ID et Client Secret → "Connecter"

Fonctionnement

  • Création d'un RDV dans Synoria → événement créé automatiquement dans Google Calendar
  • Modification d'un RDV → événement mis à jour
  • Suppression d'un RDV → événement supprimé
  • Fonctionnement offline : si pas de connexion, l'erreur est silencieuse et le RDV local reste intact
💡 Gratuit : L'API Google Calendar est dans le niveau gratuit permanent (1 million de requêtes/jour). Aucun frais même avec un usage intensif.

14. Version clé USB

La version portable fonctionne sans installation sur n'importe quel PC Windows.

Contenu du dossier clé USB

📄 Synoria.exe → L'application (lancer directement) 📁 data\ → Vos données (NE PAS SUPPRIMER) 📄 LIRE_MOI.txt → Instructions rapides 📄 Guide_Utilisation… → Ce guide complet

Utilisation

1
Branchez la clé USB sur n'importe quel PC Windows 10/11
2
Ouvrez le dossier de la clé USB
3
Double-cliquez sur Synoria.exe
✅ Aucune installation requise  ·  ✅ Données sur la clé USB  ·  ✅ Fonctionne sur plusieurs PC
⚠️ Ne jamais supprimer le dossier data\ — il contient votre base de données chiffrée.

Mise à jour de la version clé USB

  1. Téléchargez le nouveau Synoria.exe
  2. Copiez-le sur la clé USB en remplacement de l'ancien
  3. Le dossier data\ est conservé intact

16. Questions fréquentes

J'ai oublié mon mot de passe, que faire ?

Sans mot de passe, les données chiffrées sont irrécupérables. Si vous avez une sauvegarde créée avant l'oubli, vous pouvez la réimporter après avoir réinstallé l'application et créé un nouveau mot de passe. Contactez le support pour toute question : jeanpierre.timoner.mtc@gmail.com

L'application ne démarre pas / écran blanc

  • Fermez complètement et relancez
  • Essayez "Exécuter en tant qu'administrateur" (clic droit sur l'icône)
  • Vérifiez que Windows est à jour
  • Désactivez temporairement l'antivirus pour tester

Comment transférer mes données d'un PC à un autre ?

  1. Sur l'ancien PC : cliquez "💾 Sauvegarder" → notez le dossier de destination
  2. Copiez le fichier .enc (et le fichier encryption.key du dossier userData) sur une clé USB
  3. Sur le nouveau PC : installez Synoria, créez un mot de passe, puis importez la sauvegarde via "📂 Importer"

Où sont stockées mes données ?

Par défaut : C:\Users\[votre nom]\AppData\Roaming\Synoria\

Version clé USB : dossier data\ à côté de l'exécutable

L'antivirus signale le fichier

C'est un faux positif courant pour les logiciels indépendants non signés numériquement. Ajoutez une exception dans votre antivirus. Le logiciel est développé localement et ne contient aucun code malveillant.

Peut-on utiliser Synoria sur Mac ?

Le code source est compatible Mac, mais aucune version Mac n'est compilée à ce jour. Contactez le support si vous en avez besoin.

Les données sont-elles envoyées sur Internet ?

Non, sauf si vous activez la synchronisation Google Calendar. Dans ce cas, uniquement le titre "Consultation" et l'horaire du RDV sont envoyés — aucun nom de patient ni donnée médicale.
