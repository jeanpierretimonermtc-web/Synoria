Avant de désinstaller — Sauvegarder vos données
⚠️ Important — Sauvegardez avant tout

La désinstallation du logiciel ne supprime pas vos données (base de données, sauvegardes, paramètres). Mais si vous souhaitez les effacer également, faites une sauvegarde exportée au préalable au format .json.enc.

1

Exporter une sauvegarde complète

Dans Synoria, cliquez sur le bouton Sauvegarder (en-tête, bouton vert) pour créer une sauvegarde chiffrée au format .json.enc. Notez l'emplacement du fichier sauvegardé.

2

Fermer Synoria

Fermez complètement le logiciel avant de désinstaller. À la fermeture, Synoria rechiffre automatiquement la base de données — opération indispensable pour la sécurité de vos données.

2
Désinstallation sur Windows
1

Méthode 1 — Via le Panneau de configuration

  • Ouvrez Paramètres Windows → Applications → Applications installées
  • Recherchez "Synoria" dans la liste
  • Cliquez sur les trois points → Désinstaller
  • Confirmez la désinstallation
2

Méthode 2 — Via le raccourci menu Démarrer

  • Ouvrez le menu Démarrer → cherchez le dossier Synoria
  • Cliquez sur Désinstaller Synoria
  • Ou utilisez le fichier Uninstall Synoria.exe dans le dossier d'installation
✅ Ce qui est supprimé par le désinstalleur

Le désinstalleur supprime uniquement les fichiers du logiciel dans %LOCALAPPDATA%\Programs\Synoria\ ainsi que les raccourcis Bureau et menu Démarrer. Vos données sont conservées.

3
Supprimer les données sur Windows

Les données de Synoria sont stockées séparément du logiciel. Pour les supprimer entièrement :

%APPDATA%\Synoria\ # dossier principal des données
├── auth.json # vérificateur mot de passe
├── settings.json # paramètres
├── active.plugin.json # plugin actif
├── encryption.key # clé sauvegardes
├── database/
│ ├── mtc.sqlite.enc # base de données chiffrée
└── exports/ # exports Excel, JSON, registre RGPD
1

Ouvrir le dossier de données

Appuyez sur Windows + R, tapez %APPDATA%\Synoria et appuyez sur Entrée. L'Explorateur s'ouvre sur le dossier de données.

2

Supprimer le dossier

Sélectionnez tout le contenu (Ctrl+A) et supprimez (Suppr). Ou supprimez directement le dossier Synoria. Les données sont définitivement effacées.

⚠️ Action irréversible

La suppression du dossier %APPDATA%\Synoria\ est définitive. Seule une sauvegarde .json.enc préalablement exportée permettrait de restaurer vos données.

4
Désinstallation sur macOS
1

Supprimer l'application

  • Ouvrez le Finder → Applications
  • Faites glisser Synoria.app vers la Corbeille
  • Ou clic droit → Placer dans la corbeille
  • Videz la Corbeille
2

Supprimer les données (optionnel)

Les données sont dans :

  • ~/Library/Application Support/Synoria/

Ouvrez le Finder → menu Aller → Aller au dossier → tapez le chemin ci-dessus → supprimez le dossier Synoria.

5
Réinstallation après désinstallation
✅ Restauration des données

Si vous avez conservé une sauvegarde .json.enc, réinstallez Synoria, créez un nouveau mot de passe, puis importez la sauvegarde via le bouton Importer (en-tête, bouton amber). Toutes vos données seront restaurées.
