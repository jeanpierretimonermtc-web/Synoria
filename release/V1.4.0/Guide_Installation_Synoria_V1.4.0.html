Installation Windows (recommandée)
1

Lancer l'installateur

Double-cliquez sur Synoria Setup 1.4.0.exe dans le dossier Windows/.

Si Windows affiche "Éditeur inconnu" (SmartScreen), cliquez sur Plus d'informations → Exécuter quand même.

2

Suivre l'assistant d'installation

  • L'installation est silencieuse (one-click) — aucune option à saisir
  • Le logiciel est installé dans C:\Users\[Vous]\AppData\Local\Programs\Synoria\
  • Un raccourci est créé sur le Bureau et dans le menu Démarrer
3

Premier lancement

Lancez Synoria depuis le raccourci Bureau. Au premier démarrage, il vous sera demandé de créer un mot de passe qui chiffrera votre base de données.

⚠️ Conservez ce mot de passe dans un endroit sûr — il ne peut pas être récupéré.

✅ Mise à jour depuis V1.3.x

Installez directement par-dessus l'ancienne version. Vos données, paramètres et sauvegardes sont conservés dans %APPDATA%\Synoria\ et ne sont pas touchés par l'installateur.

3
Installation macOS
1

Ouvrir le DMG

Double-cliquez sur Synoria-1.4.0-universal.dmg. Une fenêtre Finder s'ouvre avec l'icône Synoria et un raccourci vers le dossier Applications.

2

Glisser dans Applications

Faites glisser l'icône Synoria.app vers le dossier Applications. Attendez la fin de la copie, puis éjectez le DMG.

3

Autoriser l'application (première fois)

  • macOS peut afficher "impossible d'ouvrir car l'identité du développeur ne peut pas être confirmée"
  • Allez dans Réglages Système → Confidentialité et sécurité → cliquez sur Ouvrir quand même
  • L'avertissement ne s'affiche qu'une seule fois
ℹ️ DMG Universel

Le fichier DMG contient une version universelle qui fonctionne nativement sur les Macs Intel (x64) et Apple Silicon (M1/M2/M3/M4) sans émulation Rosetta.

4
Version portable (Windows)
✅ Aucune installation requise

La version portable Synoria 1.4.0.exe peut être lancée depuis n'importe quel emplacement : Bureau, dossier Téléchargements, clé USB. Aucun droit administrateur nécessaire.

1

Utilisation standard

Double-cliquez sur Synoria 1.4.0.exe. Les données sont stockées dans %APPDATA%\Synoria\ (même emplacement que la version installée).

5
Version clé USB
1

Préparer la clé USB

Copiez le contenu du dossier CleUSB/ à la racine de votre clé USB (ou dans un sous-dossier Synoria).

  • Synoria 1.4.0.exe — l'exécutable portable
  • data/ — créé automatiquement au premier lancement (contient la base de données)
2

Lancer depuis la clé USB

Branchez la clé USB, naviguez vers le dossier Synoria et double-cliquez sur l'exe. Les données seront stockées sur la clé USB dans data/, pas sur l'ordinateur hôte.

⚠️ Éjection de la clé USB

Toujours fermer Synoria avant d'éjecter la clé USB. Le logiciel rechiffre la base de données à la fermeture — une éjection brusque pourrait corrompre les données.

6
Installation des plugins
1

Accéder aux paramètres

Dans Synoria, cliquez sur ⚙️ Paramètres dans la sidebar, puis sur l'onglet Plugin.

2

Importer un fichier plugin

Cliquez sur Importer un plugin (.json) et sélectionnez le fichier dans le dossier Plugins/ :

  • mtc.plugin.json — Médecine Traditionnelle Chinoise (formulaire intégré complet)
  • kinesio.plugin.json — Kinésiologie (formulaire d'anamnèse kinésiologique)
