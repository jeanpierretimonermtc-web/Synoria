================================================================
   DOSSIER PATIENT MTC — Version Clé USB
   Médecine Traditionnelle Chinoise
================================================================

CONTENU DE CE DOSSIER
─────────────────────
  📄 Dossier Patient MTC.exe   → L'application (double-cliquer pour lancer)
  📁 data\                     → Vos données patients (base de données)
  📄 LIRE_MOI.txt              → Ce fichier

COMMENT UTILISER
─────────────────
  1. Branchez la clé USB sur n'importe quel ordinateur Windows
  2. Ouvrez le dossier sur la clé USB
  3. Double-cliquez sur "Dossier Patient MTC.exe"
  4. L'application se lance — vos données sont chargées automatiquement

  ✅ Aucun logiciel à installer sur le PC hôte
  ✅ Aucun compte ou connexion internet requis
  ✅ Toutes vos données restent sur la clé USB

SAUVEGARDE DES DONNÉES
───────────────────────
  Vos données sont stockées dans le dossier  data\  sur cette clé USB.
  Ne supprimez jamais ce dossier.

  Utilisez le bouton "💾 Sauvegarder" dans l'application pour créer
  une copie de sauvegarde JSON supplémentaire.

  Il est recommandé de faire une sauvegarde régulière :
    → bouton "💾 Sauvegarder" → copie dans Documents ou un autre support

  En cas de perte de la clé, vous pourrez réimporter votre sauvegarde
  via le bouton "📂 Importer" depuis un autre ordinateur.

TRANSFERT DEPUIS UN PC (déplacer les données existantes)
──────────────────────────────────────────────────────────
  Si vous avez déjà des patients sur un PC (version installée), exportez
  une sauvegarde JSON depuis l'appli installée ("💾 Sauvegarder"), puis
  importez-la dans la version clé USB via "📂 Importer".

PROBLÈMES COURANTS
───────────────────
  • L'application ne démarre pas :
      → Vérifiez que Windows est à jour (Windows 10 64-bit ou supérieur requis)
      → Essayez de clic-droit > "Exécuter en tant qu'administrateur"
      → Windows Defender peut bloquer : cliquer "Informations complémentaires"
        puis "Exécuter quand même"

  • Fenêtre blanche au démarrage :
      → Fermez l'application et relancez-la

  • L'antivirus bloque le fichier :
      → Ce comportement est normal pour les exécutables non signés
      → Ajoutez une exception dans votre antivirus pour ce fichier
