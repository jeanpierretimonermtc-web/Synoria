Installation Windows

Fichier d'installation

📦
Synoria_Setup_1.4.3.exe
Taille : environ 85 Mo · Installateur NSIS (Windows 10 / 11, 64 bits)
Installe Synoria dans C:\Program Files\Synoria\ avec raccourci bureau et menu Démarrer

Procédure d'installation

1
Téléchargez Synoria_Setup_1.4.3.exe Ouvrez votre dossier Téléchargements dans l'Explorateur Windows et localisez le fichier.
2
Double-cliquez sur le fichier pour lancer l'installation Windows peut afficher une alerte de sécurité — c'est normal pour tout logiciel indépendant.
Alerte SmartScreen Windows
Cliquez "Informations complémentaires"
puis "Exécuter quand même"
Contrôle de compte utilisateur
Cliquez "Oui" pour autoriser l'installation
Ces alertes sont normales. Elles apparaissent pour tout logiciel non signé par un certificat commercial (coût annuel de 300-500 €). Synoria ne contient aucun code malveillant.
3
Suivez l'assistant d'installation L'assistant vous propose de choisir le dossier d'installation et de créer un raccourci sur le bureau. Les options par défaut conviennent à la grande majorité des utilisateurs.
4
Installation terminée Un raccourci Synoria apparaît sur le bureau. L'application se lance automatiquement ou vous pouvez double-cliquer sur le raccourci.
5
Premier lancement : écran de création du mot de passe Voir la section 4 de ce guide pour les instructions détaillées.

3
Version portable (clé USB)
Avantage de la version portable : Aucune installation requise. Branchez la clé USB sur n'importe quel PC Windows et lancez directement. Vos données voyagent avec vous, stockées dans le dossier data\ à côté de l'exécutable.

Préparation de la clé USB

1
Copiez le dossier Synoria entier sur la clé USB Le dossier doit contenir l'exécutable et, le cas échéant, les fichiers de données existants. Ne copiez pas le fichier seul : le dossier parent est indispensable pour que les données soient créées au bon endroit.
2
Structure du dossier sur la clé
Synoria\ ├── Synoria.exe ← application portable ├── data\ ← créé au 1er lancement │ ├── database\mtc.sqlite.enc ← base chiffrée │ ├── auth.json ← vérificateur mot de passe │ ├── encryption.key ← clé sauvegardes │ └── settings.json ← paramètres ├── GUIDE_INSTALLATION_V1.4.3.html └── LIRE_MOI.txt
3
Double-cliquez sur Synoria.exe pour lancer l'application Au premier lancement, le dossier data\ est créé automatiquement à côté de l'exécutable. Vos données seront toujours stockées dans ce dossier.
Ne supprimez jamais le dossier data\ — il contient votre base de données chiffrée et les clés nécessaires à son déchiffrement. Sans ce dossier, toutes vos données seront définitivement perdues.

Utilisation sur un autre PC

1
Branchez la clé USB sur le nouveau PC Windows.
2
Ouvrez l'Explorateur de fichiers → naviguez vers votre clé USB → dossier Synoria.
3
Double-cliquez sur Synoria.exe — vos données et votre mot de passe sont chargés automatiquement.
Conseil : Pour protéger votre clé USB en cas de perte, activez le chiffrement de la clé (BitLocker sur Windows ou VeraCrypt) en plus du mot de passe Synoria. Cela offre une double couche de protection pour les données de vos patients.

Premier lancement — Création du mot de passe

Au tout premier démarrage de Synoria, l'application affiche l'écran de configuration du mot de passe. Cette étape est obligatoire et irréversible.

Sans lui, vos données seront définitivement illisibles.

🔐 Créer le mot de passe
1
Choisissez un mot de passe robuste Minimum 8 caractères. Recommandé : au moins 12 caractères combinant majuscules, minuscules, chiffres et caractères spéciaux (ex. : MonCabinet@2026!).
2
Confirmez le mot de passe en le saisissant une seconde fois dans le champ de confirmation.
3
Cochez la case de confirmation En cochant cette case, vous attestez avoir noté votre mot de passe en lieu sûr. L'application ne peut pas le récupérer pour vous.
4
Cliquez sur "Créer le mot de passe" La base de données est chiffrée en AES-256. L'application s'ouvre automatiquement sur le tableau de bord.
5
Activez les sauvegardes automatiques immédiatement Rendez-vous dans Paramètres → Sauvegardes et configurez un dossier de sauvegarde automatique. Ne remettez pas cette étape à plus tard.
IMPORTANT — Mot de passe irrécupérable :
Le chiffrement AES-256 utilisé par Synoria est conçu pour être indéchiffrable sans le mot de passe. Il n'existe aucune procédure de récupération. Si vous oubliez votre mot de passe, vos données seront définitivement inaccessibles.

Que faire dès maintenant :
  • Notez votre mot de passe dans un gestionnaire de mots de passe (Bitwarden, 1Password, KeePass).
  • Ou écrivez-le sur papier et conservez-le dans un endroit physiquement sécurisé (coffre, tiroir verrouillé).
  • Activez les sauvegardes automatiques dans les paramètres.
Connexions suivantes : À chaque démarrage, Synoria affiche un écran de connexion. Saisissez votre mot de passe et cliquez "Déverrouiller". Après 20 minutes d'inactivité, l'application se verrouille automatiquement pour protéger vos données.

Mise à jour

La mise à jour de Synoria est simple et sans risque pour vos données. Les mises à jour incluent de nouvelles fonctionnalités, des améliorations de performance et des correctifs de sécurité.

1
Téléchargez le nouvel installateur Récupérez le fichier Synoria_Setup_X.X.X.exe fourni par votre contact Synoria ou depuis le lien reçu par e-mail.
2
Lancez le nouvel installateur par-dessus l'ancienne version Il n'est pas nécessaire de désinstaller la version précédente. Le nouvel installateur met à jour les fichiers de l'application tout en laissant vos données intactes.
3
Suivez l'assistant — acceptez toutes les alertes SmartScreen Identique à l'installation initiale. Cliquez "Informations complémentaires" → "Exécuter quand même" si l'alerte apparaît.
4
Relancez Synoria Vos données, votre mot de passe et tous vos paramètres sont conservés automatiquement. Aucune action supplémentaire n'est requise.

Ce qui est conservé lors d'une mise à jour

  • Tous les dossiers patients et l'historique des séances
  • Votre mot de passe et le chiffrement de la base de données
  • Vos paramètres (sauvegardes, facturation, RGPD)
  • Les rendez-vous du calendrier
  • Les plugins actifs
  • L'historique des sauvegardes et les clés de chiffrement
Recommandation : Effectuez une sauvegarde manuelle (bouton sauvegarde dans les Paramètres) avant toute mise à jour majeure. C'est une précaution facultative mais bonne pratique.
6
Désinstallation

Désinstaller Synoria (Windows)

1
Ouvrez Paramètres Windows → Applications Cliquez sur le menu Démarrer → icône Paramètres (roue dentée) → Applications → Applications installées.
2
Recherchez "Synoria" dans la liste Cliquez sur les trois points à droite de l'entrée Synoria → Désinstaller → confirmez.
3
L'application est supprimée Les fichiers dans C:\Program Files\Synoria\ sont supprimés, ainsi que les raccourcis.
Vos données sont conservées : La désinstallation supprime uniquement l'application, pas vos données patients. Celles-ci restent dans %APPDATA%\Synoria\. Si vous souhaitez effacer définitivement toutes vos données, supprimez manuellement ce dossier après la désinstallation.

Sauvegardez avant de supprimer vos données

Si vous désinstallez définitivement Synoria et souhaitez conserver une copie de vos dossiers patients (par exemple pour archivage RGPD), effectuez une export complet depuis Paramètres → Sauvegardes → Exporter avant de supprimer le dossier %APPDATA%\Synoria\.

En cas de problème

L'installation est bloquée par l'antivirus ou SmartScreen

Cause : Certains antivirus signalent les exécutables non signés numériquement comme suspects (faux positif). Synoria ne contient aucun logiciel malveillant.

Solution : Lors de l'alerte SmartScreen, cliquez "Informations complémentaires" puis "Exécuter quand même". Si votre antivirus (Windows Defender, Avast, etc.) bloque le fichier, ajoutez-le en exclusion : Sécurité Windows → Protection contre les virus → Exclusions → Ajouter → Fichier → sélectionnez Synoria_Setup_1.4.3.exe.

L'application affiche un écran blanc ou ne démarre pas

Solution 1 : Fermez complètement l'application et relancez-la (parfois nécessaire au tout premier démarrage).

Solution 2 : Clic droit sur le raccourci Synoria → "Exécuter en tant qu'administrateur".

Solution 3 : Vérifiez que Windows est à jour via Windows Update (Paramètres → Windows Update → Vérifier les mises à jour).

Le mot de passe est refusé après une réinstallation

Cause : Le dossier de données existant (%APPDATA%\Synoria\) contient les fichiers d'une installation précédente avec un mot de passe différent.

Solution : Ouvrez l'Explorateur Windows → barre d'adresse → tapez %APPDATA%\Synoria → Entrée. Vérifiez que les fichiers auth.json et database\mtc.sqlite.enc correspondent bien à votre installation. Si vous avez une sauvegarde récente, restaurez-la depuis Paramètres → Sauvegardes → Importer.

La version portable (clé USB) ne reconnaît pas les données

Cause : L'exécutable Synoria.exe a été copié seul, sans le dossier data\ qui le contenait.

Solution : Assurez-vous que Synoria.exe et le dossier data\ se trouvent bien dans le même dossier parent sur la clé USB. L'exécutable portable cherche ses données dans un sous-dossier data\ situé à côté de lui.

Générer un rapport de diagnostic

Pour tout problème persistant, Synoria peut générer un rapport de diagnostic contenant les informations techniques nécessaires à l'assistance :

1
Dans Synoria, accédez à Paramètres (icône engrenage dans la barre de navigation).
2
Cliquez sur l'onglet "Support" puis sur le bouton "Générer le rapport de diagnostic".
3
Enregistrez le fichier et joignez-le à votre e-mail de support.
Note : Le rapport de diagnostic ne contient aucune donnée patient. Il inclut uniquement la version du logiciel, les informations système (OS, version), les paramètres de configuration non sensibles et les journaux d'erreurs techniques.
