V 1.4.3

Panneau administrateur, diagnostic & documentation

Juin 2026
✦ Actuelle
✨ Nouvelles fonctionnalités
Panneau administrateur caché (Ctrl + Shift + Alt + A) Interface sécurisée par mot de passe réservée au créateur du logiciel. 4 onglets : Système (version, OS, RAM), Logs (journal de session), Base de données (intégrité, stats), Configuration (paramètres bruts). Accessible depuis n'importe quelle page, sans connexion réseau.
Procédure "Mot de passe oublié" Lien cliquable sur l'écran de connexion. Génère un document HTML avec la procédure complète de récupération, adapté à l'environnement de l'utilisateur.
Outil de diagnostic avancé Rapport HTML complet (système, base de données, derniers logs) exportable depuis les Paramètres. Facilite le support technique à distance.
Guides utilisateur intégrés Guide d'installation, guide d'utilisation complet, protocole de récupération MDP, procédure d'envoi des logs — tous disponibles dans le dossier release.
Build macOS via GitHub Actions Workflow CI automatique : chaque nouvelle version déclenche la compilation d'un DMG universel (Intel + Apple Silicon) sur serveur Mac distant.
⚡ Améliorations
Identité visuelle unifiée Toutes les références "Dossier Patient MTC" remplacées par "Synoria" dans l'interface, les documents et les métadonnées.
Code source allégé Suppression du code de migration de l'ancienne application (aucun utilisateur pré-1.4). Démarrage plus rapide et code plus maintenable.
1 panneau admin 9 canaux IPC admin 4 guides utilisateur Build Mac CI Diagnostic avancé

V 1.4.2

Templates, recherche globale, mode sombre & raccourcis

Mai 2026
Précédente
✨ Nouvelles fonctionnalités
Templates de séances Sauvegarde et réutilisation de modèles de séances. Gain de temps pour les consultations récurrentes avec le même protocole.
Recherche globale Barre de recherche unifiée permettant de retrouver un patient, une séance ou un rendez-vous depuis n'importe quelle page.
Mode sombre complet Thème sombre appliqué à l'intégralité de l'interface, y compris les modales, formulaires de séance et module comptabilité.
Raccourcis clavier Naviguer dans l'application sans souris. Raccourcis pour les actions fréquentes : nouvelle séance, nouveau patient, sauvegarde rapide.

V 1.4.1

Tableau de bord enrichi, calendrier & profil praticien

Avril 2026
Archivée
✨ Nouvelles fonctionnalités
Tableau de bord enrichi Nouvelles statistiques : taux d'activité mensuel, évolution du nombre de patients, revenus comparatifs. Graphiques interactifs intégrés.
Améliorations du calendrier Navigation améliorée entre les vues. Meilleure gestion des conflits de créneaux. Affichage du nom du patient directement dans la grille horaire.
Module comptabilité renforcé Export Excel enrichi avec les catégories de dépenses. Calcul URSSAF mis à jour. Gestion des types de consultations personnalisables.
Profil praticien Paramétrage du nom, spécialité et coordonnées du praticien, utilisés automatiquement dans les factures et exports PDF.
Landing page Synoria hébergée sur Vercel Page de présentation du logiciel avec liens de téléchargement vers les releases GitHub. Favicon Synoria dans l'onglet navigateur. Responsive 4 breakpoints.

V 1.4.0

Refonte de l'interface & calendrier 3 vues

Mars 2026
Fondation 1.4.x
🚀 Version fondatrice de la série 1.4.x
Refonte complète de l'interface utilisateur Nouvelle identité visuelle Synoria. Palette de couleurs cohérente, typographie affinée, navigation latérale repensée. Transition de "Dossier Patient MTC" vers la marque Synoria.
Calendrier 3 vues Vue mensuelle (planning global), vue hebdomadaire (organisation de la semaine), vue journalière (créneaux de 30 min de 08h00 à 19h30).
Compatibilité macOS Corrections de compatibilité pour macOS : chemins de fichiers, menu Édition natif, raccourcis clavier, fenêtre toujours visible.
Google Calendar Synchronisation bidirectionnelle OAuth2 avec Google Agenda (lecture, création, mise à jour et suppression de rendez-vous).
Sécurité renforcée Rollback automatique de auth.json si l'initialisation de la base échoue pendant la création du mot de passe. Évite les états incohérents.
Refonte UI complète Calendrier 3 vues Google Calendar Compat. macOS
