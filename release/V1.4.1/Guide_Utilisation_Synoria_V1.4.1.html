Vue d'ensemble

Synoria est un logiciel de bureau pour la gestion de dossiers patients de praticiens de santé. Toutes les données sont stockées localement sur votre ordinateur, chiffrées avec l'algorithme AES-256-GCM, et ne quittent jamais votre machine. Il n'y a pas de connexion internet, pas de cloud, pas d'abonnement.

4
Gestion des patients

Créer un patient

1

Ouvrir le formulaire

Cliquez sur + Nouveau patient en haut à droite de la page Patients.

2

Renseigner les informations

  • Obligatoires : Nom, Prénom
  • Recommandés : Date de naissance, Téléphone
  • Optionnels : Email, Adresse, Profession, Médecin traitant, Médicaments, Antécédents, Alertes, Notes générales
3

Consentement RGPD

Cochez la case Consentement donné si le patient a signé votre formulaire de consentement. La date est enregistrée automatiquement.

Actions sur une fiche patient

ActionDescription
+ SéanceOuvre le formulaire Nouvelle séance pré-rempli avec ce patient
📋 SéancesFiltre l'historique des séances sur ce patient uniquement
🧾 FactureGénère une facture PDF pour ce patient
✏️ ModifierÉdite les informations du patient (sur survol)
💾 BackupExporte une sauvegarde de ce seul patient (sur survol)
📦 ArchiverMasque le patient de la liste active (sur survol)
🗑 SupprimerSuppression définitive avec confirmation (sur survol)
⚠️ Archiver vs Supprimer

Privilégiez l'archivage à la suppression pour les patients inactifs. Un patient archivé conserve tout son historique et peut être réactivé à tout moment. La suppression est irréversible.

5
Créer et rédiger une séance
1

Sélectionner le patient

Cliquez sur Nouvelle séance dans la sidebar. Sélectionnez le patient dans la liste déroulante ou via "+ Séance" directement depuis sa fiche.

2

Remplir les sections

Les sections disponibles dépendent du plugin actif :

  • Sans plugin : Motif · Évolution · Anamnèse · Traitement · Réactions
  • Plugin MTC : Prise de notes · Questionnaire systèmes · Observation · Tests énergétiques · Analyse · Traitement · Barrage
  • Autre plugin : Les sections définies dans le fichier .json du plugin
3

Barre de formatage

Dans les zones de texte enrichi (fond légèrement coloré), utilisez la barre de formatage en haut de l'application :

  • G Gras · I Italique · S Souligné
  • Listes à puces · listes numérotées
  • La barre s'active automatiquement quand le curseur est dans une zone de texte enrichi
4

Planifier la prochaine séance

Dans la section Plan de suivi (tout en bas), saisissez la date du prochain rendez-vous. Un rendez-vous est automatiquement créé dans le calendrier, ou mis à jour s'il existe déjà.

5

Enregistrer

Cliquez sur Enregistrer la séance. Vous êtes redirigé vers le résumé de la séance depuis lequel vous pouvez exporter en Excel, JSON ou imprimer.

JourFocus sur une journée, détail de tous les RDV du jour

Créer un rendez-vous

1

Clic sur "+ RDV" ou sur un créneau libre

Dans les vues Semaine ou Jour, cliquez sur un créneau horaire vide pour pré-remplir la date et l'heure. Une modal s'ouvre.

2

Renseigner le rendez-vous

  • Patient existant : sélectionnez-le dans la liste — le RDV sera lié à sa fiche
  • Invité : saisissez Nom, Prénom, Téléphone (sans créer de fiche patient)
  • Heure de début / fin · Note libre
3

Créer la séance depuis un RDV

Sur un RDV patient, le bouton "Créer la séance" navigue directement vers le formulaire Nouvelle séance avec le patient et la date pré-remplis.

ℹ️ Synchronisation automatique

Quand vous renseignez une prochaine séance dans le formulaire de saisie, un rendez-vous est automatiquement créé ou mis à jour dans le calendrier. Pas besoin de créer le RDV manuellement.

Sauvegardes

Sauvegarde manuelle

Cliquez sur le bouton Sauvegarder (vert, en-tête) à tout moment. Un fichier .json.enc chiffré est créé dans le dossier de sauvegarde configuré.

Sauvegarde automatique

Activez la sauvegarde automatique dans Paramètres → Sauvegardes :

  • À la fermeture : sauvegarde à chaque fermeture du logiciel
  • Quotidienne : sauvegarde automatique une fois par jour
  • Rétention : nombre de jours à conserver (les plus anciennes sont supprimées)

Restaurer une sauvegarde

1

Cliquer sur Importer

Bouton amber Importer en haut à droite. Sélectionnez le fichier .json.enc.

2

Confirmer la restauration

⚠️ La restauration remplace toutes vos données actuelles par celles de la sauvegarde. Un avertissement vous demande de confirmer avant de procéder.

⚠️ Format propriétaire

Les fichiers .json.enc sont chiffrés avec la clé de votre installation. Une sauvegarde créée sur l'installation A peut uniquement être restaurée sur la même installation (même fichier encryption.key).

Plugins (formulaires spécialisés)

Les plugins permettent d'adapter le formulaire de séance à votre spécialité. Deux plugins neutres sont fournis avec cette version.

PluginSpécialitéFormulaire
mtc.plugin.jsonMédecine Traditionnelle ChinoiseFormulaire MTC intégré complet (langue, pouls, 5 éléments…)
kinesio.plugin.jsonKinésiologieContexte RDV, signes psycho/physiques, motivation, objectifs

Installer un plugin

1

Paramètres → Plugin

Cliquez sur Importer un plugin, sélectionnez le fichier .json du plugin souhaité dans le dossier Plugins/.

2

Activer le plugin

Le plugin importé est immédiatement actif. Le formulaire de la page Nouvelle Séance bascule automatiquement vers le formulaire du plugin.

ℹ️ Données historiques préservées

Chaque séance enregistrée conserve une copie du schéma du plugin au moment de la saisie. Changer de plugin ne modifie pas les séances déjà enregistrées.

13
Sécurité et mot de passe
✅ Chiffrement de bout en bout

La base de données est déchiffrée au login et rechiffrée à la fermeture (AES-256-GCM). Sur le disque, seul le fichier .sqlite.enc chiffré existe — jamais en clair à l'arrêt.

FonctionnalitéDescription
Verrouillage autoAprès 20 minutes d'inactivité, l'application se verrouille
Verrouillage manuelBouton "Verrouiller" en bas de la sidebar (badge rose)
Changer le mot de passeParamètres → Mot de passe (ancien mot de passe requis)
Mot de passe oublié⚠️ Aucune récupération possible — les données sont irrémédiablement perdues
⚠️ Conservez votre mot de passe

En cas d'oubli du mot de passe, il est impossible de récupérer les données (c'est le principe même du chiffrement fort). Notez votre mot de passe dans un gestionnaire de mots de passe ou un endroit sécurisé.
